Usage Rules / 사용 규칙

✓ Do

  • Use the logo on backgrounds with sufficient contrast
  • Use approved color variations only
  • Maintain clear space requirements
  • Use gradient logo on dark backgrounds
  • Use dark/monochrome on light backgrounds

✗ Don't

  • Stretch, skew, or rotate the logo
  • Change gradient colors or direction
  • Add shadows, outlines, or effects
  • Place on busy or low-contrast backgrounds
  • Rearrange mark and wordmark relationship
  • Recreate in a different font

08 — Photography / 사진 스타일
Photography Style
Photography should feel authentic, global, and professional. Focus on real manufacturing environments, diverse teams, and international trade settings.

🌎

✓ Do

  • Show real manufacturing environments
  • Feature diverse, global teams
  • Use natural lighting when possible
  • Include contextual details (products, shipping)
  • Apply subtle color grading toward cool tones

✗ Don't

  • Use generic stock photos with forced smiles
  • Over-saturate or use heavy filters
  • Show cluttered or messy environments
  • Use imagery that feels exclusively Korean
  • Mix warm and cool color treatments
09 — Accessibility / 접근성
Accessibility
We target WCAG 2.2 Level AA. Inclusive design is not optional. Every user should have equal access to our platform.
Contrast Checks / 대비 검사
Cyan on Dark BG
Ratio: 10.2:1 AAA
White 90% on Dark BG
Ratio: 17.8:1 AAA
Blue on White
Ratio: 4.6:1 AA
Dark on White
Ratio: 16.2:1 AAA
White 40% on Dark BG
Ratio: 5.3:1 AA
White 20% on Dark BG
Ratio: 2.6:1 Decorative only
Requirements / 요건 사항
Focus indicators — 2px solid cyan, 2px offset
Touch targets — Minimum 44x44px
Text contrast — 4.5:1 minimum (AA)
Large text — 3:1 minimum (18px+ bold)
Motion — Respect prefers-reduced-motion
Color — Never convey info by color alone
Keyboard Navigation / 키보드 내비게이션
Tab — Navigate interactive elements
Enter/Space — Activate buttons and links
Escape — Close modals and overlays
Arrow keys — Navigate menus and tabs
Focus trap — Modal dialogs trap focus
Skip link — Skip to main content
10 — Multilingual / 다국어 지원
Multilingual Support
Whistle AI serves a global audience. Our design system accommodates Korean, English, and future language expansions with graceful typography fallbacks.
Font Stack Priority

Layout Considerations / 레이아웃 고려 사항
Line height — Korean text needs 1.8 (vs 1.6 English)
Text expansion — Korean is typically 20-30% shorter
Number formatting — Use locale-aware formatting
Date format — YYYY-MM-DD (ISO 8601)
Currency — USD for global, KRW for Korean market
