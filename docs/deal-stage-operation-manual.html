Stage 1 — 바이어 매칭
관련 역할: AI 매칭 엔진, 제조사, 바이어 | 예상 소요: 1~7일

프로세스

AI가 제조사 상품과 바이어 니즈를 분석하여 최적의 바이어를 추천하거나, 바이어가 직접 인바운드로 관심을 표현합니다.

  • AI 추천: 상품 카테고리, 목적국, 바이어 구매 이력 기반 매칭
  • 인바운드: 바이어가 상품 페이지에서 직접 문의
  • 관심 표현 수신 시 제조사에게 알림 발송

필수 서류

이 단계에서는 별도 서류가 필요하지 않습니다.

다음 단계 트리거

제조사가 바이어의 관심 표현을 수락하면 Stage 2 (협상)으로 자동 전환됩니다.

바이어 관심 표현 후 72시간 내 미응답 시 자동 만료됩니다. 제조사에게 24시간, 48시간 시점에 리마인더가 발송됩니다.
Stage 2 — 협상
관련 역할: 제조사, 바이어 | 예상 소요: 3~14일

프로세스

  • 플랫폼 내 채팅으로 가격, 수량, 납기, 품질 조건 협의
  • 샘플 요청 및 발송 관리
  • Incoterms(무역조건) 합의: FOB, CIF, EXW 등
  • 결제 조건 합의: 선금/잔금 비율, 에스크로 사용 여부

필수 서류

서류작성자비고
견적서 (Quotation)제조사단가, 수량, 총액, 무역조건 포함
샘플 송장 (Sample Invoice)제조사샘플 발송 시

다음 단계 트리거

양측이 조건에 합의하고 제조사가 "협상 완료"를 클릭하면 Stage 3 (관세사 매칭)으로 전환됩니다.

협상 중 14일간 채팅 활동이 없으면 "비활성 딜"로 분류되며, 양측에 알림이 발송됩니다. 추가 7일간 미활동 시 딜이 자동 보류됩니다.
Stage 3 — 관세사 매칭
관련 역할: AI 매칭 엔진, 관세사 | 예상 소요: 1~3일

프로세스

  • AI가 상품 카테고리, 목적국, 규제 요건 기반으로 적합 관세사 추천
  • 추천 관세사에게 딜 정보 및 수임료 안내와 함께 배정 요청
  • 관세사 수락/거절 응답 대기 (48시간)
  • 거절 또는 미응답 시 차순위 관세사에게 자동 재배정

필수 서류

서류작성자비고
상품 정보 시트제조사HS코드 참고용 상세 정보
수출 대상국 규제 요약AI 자동 생성관세사 참고용

다음 단계 트리거

관세사가 딜을 수락하면 Stage 4 (포워더 매칭)으로 전환됩니다.

수임료 안내: 관세사 수임료는 상품 카테고리와 목적국 난이도에 따라 차등 적용됩니다. 플랫폼 수수료(10%, Gold SP 8%)는 수임료에서 차감 후 정산됩니다.
Stage 4 — 포워더 매칭
관련 역할: AI 매칭 엔진, 포워더 | 예상 소요: 1~3일

프로세스

  • AI가 출발지, 목적지, 화물 규격, 운송 방식 기반으로 포워더 추천
  • 포워더에게 운임 견적 요청
  • 포워더 견적 제출 및 수락 대기 (48시간)
  • 제조사가 견적 확인 후 포워더 확정

필수 서류

서류작성자비고
화물 명세서제조사중량, 부피, 포장 규격
운임 견적서포워더운송 방식별 견적 (해상/항공/특송)

다음 단계 트리거

제조사가 포워더 견적을 승인하면 Stage 5 (생산/검사)로 전환됩니다.

Stage 5 — 생산/검사
관련 역할: 제조사 | 예상 소요: 7~30일

프로세스

  • 생산 일정 등록 및 진행률 업데이트
  • 중간 검사 체크리스트 작성 (해당 시)
  • 사전 선적 검사 (Pre-Shipment Inspection) 진행
  • 바이어에게 생산 진행 상황 자동 공유

필수 서류

서류작성자비고
생산 일정표제조사예상 완료일 포함
검사 보고서제조사/검사기관품질 기준 충족 여부
카테고리별 인증서제조사식품: HACCP, 화장품: GMP 등

다음 단계 트리거

생산 완료 및 검사 통과 확인 후 제조사가 "생산 완료"를 클릭하면 Stage 6 (결제)으로 전환됩니다.

생산 일정이 당초 계획 대비 7일 이상 지연될 경우 바이어에게 자동 알림이 발송되며, 바이어는 딜 취소를 요청할 수 있습니다.
Stage 6 — 결제
관련 역할: 바이어, 플랫폼 (Stripe) | 예상 소요: 1~3일

프로세스

  • Proforma Invoice (PI) 발행 — 최종 금액, 결제 조건 확정
  • 바이어 에스크로 결제 (Stripe)
  • 결제 확인 후 자금 에스크로 보관
  • 결제 완료 알림 → 제조사, 포워더, 관세사에게 동시 발송

필수 서류

서류작성자비고
Proforma Invoice제조사상품 상세, 금액, 결제 조건
결제 확인서플랫폼 자동 생성에스크로 보관 확인

에스크로 자금 흐름

3단계 자금 방출:
1. 주문 수락 확인 (24시간 내) → 자금 홀드 유지
2. 출하 확인 → 50% 캡처 (제조사에게 선지급)
3. 수취 확인 → 나머지 50% 캡처 (최종 정산)

14일 무응답 시 자동 수취 확인 처리

다음 단계 트리거

결제 완료 확인 시 Stage 7 (선적/통관)으로 자동 전환됩니다.

Stage 7 — 선적/통관
관련 역할: 포워더, 관세사, 제조사 | 예상 소요: 3~14일

프로세스

  • 관세사: 수출신고서 작성 및 세관 제출
  • 관세사: HS코드 확정, 관세율 확인, 원산지증명 발급
  • 포워더: 선적 예약 (Booking), 컨테이너/항공 화물 배정
  • 포워더: B/L(선하증권) 발급
  • 수출신고 수리 확인
  • 선적 완료 후 트래킹 번호 공유

필수 서류

서류작성자비고
수출신고서관세사세관 제출용
Commercial Invoice제조사상품 가격, 수량, 원산지
Packing List제조사포장 상세, 중량, 마킹
B/L (선하증권)포워더/선사해상 운송 시
AWB (항공화물운송장)포워더/항공사항공 운송 시
원산지증명서관세사/상공회의소FTA 적용 시 필수
보험증권제조사/포워더CIF 조건 시

다음 단계 트리거

선적 완료 확인 및 B/L 발급 후 Stage 8 (도착/정산)으로 전환됩니다.

수출신고 반려 시 관세사에게 즉시 알림이 발송됩니다. 반려 사유 확인 후 24시간 내 재신고해야 합니다.
Stage 8 — 도착/정산
관련 역할: 바이어, 플랫폼 | 예상 소요: 3~7일

프로세스

  • 바이어 수취 확인 (앱/웹에서 "수취 완료" 클릭)
  • 최종 에스크로 자금 방출 (잔여 50% 캡처)
  • 플랫폼 수수료 차감 후 각 참여자에게 정산
  • 바이어 → 제조사 리뷰 작성
  • 제조사 → 관세사/포워더 리뷰 작성
  • 딜 완료 처리 및 아카이브

정산 내역

정산 대상정산 내용수수료
제조사상품 대금플랫폼 수수료 2.5% 차감
관세사수임료SP 수수료 10% (Gold 8%) 차감
포워더운임SP 수수료 5% (Gold 4%) 차감

다음 단계 트리거

바이어 수취 확인 + 전체 정산 완료 시 딜이 완료(Closed) 상태로 전환됩니다.

자동 수취 확인: 바이어가 화물 도착 후 14일간 응답하지 않을 경우, 자동으로 수취 확인 처리되며 최종 정산이 진행됩니다.
