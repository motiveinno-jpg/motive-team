## 1. EXECUTIVE SUMMARY

**Whistle AI**는 **전 세계 제조사와 바이어를 연결하는 AI 기반 글로벌 수출 통합 플랫폼**입니다.

AI가 시장분석부터 바이어 매칭, 서류 자동생성, 안전결제까지 **수출의 전 과정을 하나의 플랫폼에서** 완결합니다. 한국을 시작으로, 전 세계 제조사가 국경 없이 제품을 판매할 수 있는 인프라를 구축합니다.

> **직원 1명 급여보다 적은 금액으로, 전세계 누구나 글로벌 수출제품으로 쉽게 연결되는 서비스.**

## 2. 문제 정의 (Problem)

### 전 세계 중소 제조사의 수출 현실

글로벌 B2B 무역 시장은 $21T 규모이지만, 대부분의 중소 제조사는 수출에 접근조차 못 합니다.
한국만 봐도:

| 지표 | 수치 |
|------|------|
| 한국 중소기업 수출액 (2025) | **$118.6B** (전년 대비 +6.9%) |
| 수출 중소기업 수 (2025) | **98,219개사** |
| 전체 중소기업 중 수출 비율 | **약 1.3%** |
| 수출 포기 사유 1위 | "절차가 너무 복잡하다" |

### 한국 시장의 구조적 문제

**내수 시장은 포화 상태입니다.**
- 한국 유통 = 네이버 아니면 쿠팡 (양강 독점 구조)
- 신규 브랜드는 매년 수천 개 런칭되지만, 내수 시장은 이미 포화
- **이 제품들을 해외로 보내야 합니다. 그런데 한국에 수출 플랫폼이 없습니다.**

알리바바(중국), Amazon(미국), Faire(미국) — 모두 해외 플랫폼입니다. **한국 제조사를 위한, 한국에서 만든 글로벌 수출 플랫폼은 단 하나도 없습니다.**

이것이 Whistle AI가 존재해야 하는 이유입니다.

### 제조사의 수출 페인포인트

1. **정보 비대칭** — HS코드, FTA, 인증 요건을 파악하려면 전문가 의존 (건당 50~300만원)
2. **프로세스 분절** — 시장조사(A업체) → 서류(B업체) → 물류(C업체) → 결제(D업체) 각각 별도
3. **바이어 접근성** — 알리바바/전시회 의존, 매칭 품질 불확실
4. **비용 부담** — 수출 초기 비용 3,000만~1억원 (관세사+포워더+번역+마케팅)
5. **플랫폼 부재** — 한국에 수출 전용 플랫폼 자체가 존재하지 않음

## 3. 솔루션 (Solution)

### Whistle AI 6서비스 체인

```
AI 수출분석 → 인증 대행 → 바이어 매칭 → 서류 자동생성 → 안전결제 → 물류/통관
```

| 서비스 | 설명 | 기존 방식 대비 |
|--------|------|---------------|
| **AI 수출분석** | 상품 URL/이미지 → AI가 HS코드 분류, FTA 시뮬레이션, 시장분석, 마진 계산까지 자동 생성 | 관세사 의뢰 2주 → **5분** |
| **인증 대행** | 수출 대상국별 필요 인증(FDA, CE, KC 등) 자동 파악 + 진행 추적 | 직접 조사 → **자동 매핑** |
| **바이어 매칭** | AI가 제품 특성 기반 최적 바이어 추천 + 채팅 | 전시회 참가 500만원 → **월 $99~** |
| **서류 자동생성** | PI/CI/PL/CO/SC 등 수출 서류 자동 생성 + 인라인 원가계산기 | 건당 30~50만원 → **무제한** |
| **안전결제** | 에스크로 기반 3단계 자금 방출 (주문수락→출하확인→수취확인) | 신뢰 불확실 → **플랫폼 보증** |
| **물류/통관** | 실시간 배송 추적 + 관세 시뮬레이션 | 포워더 개별 계약 → **통합 관리** |

## 4. 기술 (Technology)

### AI 분석 엔진 v7 — 핵심 기술 스택

| 기술 영역 | 상세 |
|-----------|------|
| **멀티모달 AI 분석** | Claude Sonnet + Vision API — 이미지/URL 직접 읽어 상품 특성 추출 |
| **3인 전문가 패널** | 관세사(15년)/물류전문가(20년)/수출실무(12년) 시뮬레이션 합의 |
| **실시간 데이터** | UN Comtrade, 한국 관세청, World Bank, 실시간 환율 API |
| **HS코드 자동 분류** | 10자리 HS코드 자동 매칭 (Edge Function) |
| **FTA 시뮬레이터** | 21개 FTA 협정, 59개국 관세율 시뮬레이션 |
| **비용/마진 계산기** | 5탭 통합: HS분류/FTA분석/원가계산/물류비/마진 시뮬레이션 |
| **에스크로 결제** | Stripe manual capture — 3단계 자금 방출 |

### 아키텍처

- **Frontend:** Vanilla JS SPA (프레임워크 없음 → 초경량, 빠른 로딩)
- **Backend:** Supabase (Auth + PostgreSQL + RLS + Realtime + Edge Functions)
- **결제:** Stripe (글로벌 통합)
- **호스팅:** Cloudflare Pages (글로벌 CDN)
- **보안:** RLS 100% 적용, Rate Limiting, XSS 보호

### 데이터베이스 — 14 테이블

```
users, companies, products, analyses, projects, buyers, matchings,
documents, service_requests, voucher_records, search_logs,
contact_requests, shipping_rates, tariff_rates
```
