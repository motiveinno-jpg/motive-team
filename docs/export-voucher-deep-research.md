## 1. 수출바우처 사업 개요

### 1.1 사업 정의

수출바우처 사업(정식명칭: 수출지원기반활용사업)은 **중소/중견기업이 자사의 수출역량에 맞는 수출지원 서비스를 자유롭게 선택할 수 있도록 바우처(쿠폰) 형태로 정부가 국고를 지원하는 사업**이다.

- 기업이 일정 금액을 자부담하면, 정부가 그에 상응하는 국고를 매칭하여 온라인 쿠폰 형태의 수출바우처를 발급
- 기업은 바우처 한도 내에서 14개 분야 7,500여개 서비스를 자유롭게 선택하여 수행 후 소요비용 정산

### 1.2 2026년 예산 규모

| 구분 | 금액 |
|------|------|
| **총 예산** | **1,502억원** (2025년 대비 226억원 증가) |
| 산업부 산업바우처 | 474억원 (전년 대비 약 30% 증가) |
| 중기부 수출바우처 | 약 1,028억원 |
| 모집 기업 수 | 약 2,000개사 (1차 기준) |

### 1.3 운영 체계

| 역할 | 기관 |
|------|------|
| **주관부처** | 산업통상자원부, 중소벤처기업부 |
| **총괄운영기관** | KOTRA(산업부 소관), 중소벤처기업진흥공단/KOSME(중기부 소관) |
| **수행기관** | 민간 서비스 제공 업체 (현재 **2,246개 기관** 등록) |
| **참여기업** | 수출 희망 중소/중견기업 |

### 1.4 사업 기간 (2026년)

| 구분 | 기간 |
|------|------|
| 중기부 수출바우처 | 2026.02.01 ~ 2027.01.31 |
| 산업부 산업바우처 | 2026.02.01 ~ 2027.01.31 |

## 3. 참여기업(수혜기업) 관점

### 3.1 신청 자격 요건

**기본 자격:**
- 중소기업기본법상 중소기업에 해당하는 법인 및 개인사업자
- 중견기업 특별법상 중견기업
- **사업자등록 후 1년 이상 경과 필수**

**제외 대상:**
- 휴/폐업 중인 사업자
- 국세/지방세 체납자
- 과거 바우처 부정사용 이력 기업
- 동일 대표자 중복 신청
- 바우처 사용률 70% 미만 기업은 **2년간 신청 불가**
- 자본잠식 상태 기업 (심사 탈락 가능성 높음)

### 3.2 트랙 분류 및 지원 금액 (2026년 기준)

| 트랙 | 수출 실적 기준 | 바우처 한도 | 자부담 비율 |
|------|---------------|------------|------------|
| **내수기업** | 수출 실적 없음 (매출 10억+ 필요) | **3,000만원** | 중소 30% / 중견 50% |
| **초보기업** | 최근 3년 수출 10만불 미만 | **3,000만원** | 중소 30% / 중견 50% |
| **유망기업** | 수출 10만~100만불 | **4,500만원** | 중소 30% / 중견 50% |
| **성장기업** | 수출 100만~500만불 | **7,000만원** | 중소 30% / 중견 50% |
| **강소기업** | 수출 500만불 초과 | **1억원** | 중소 30% / 중견 50% |
| **지정기업** | 브랜드K, 글로벌강소기업 등 | 별도 | 별도 |

**추가 지원:** 관세대응 패키지 활용 기업에 **최대 5,000만원 추가** (최대 1.5억원)

### 3.3 기업 자부담 구조

| 기업 유형 | 정부 지원 | 기업 부담 | 예시 (바우처 1억원) |
|-----------|----------|----------|-------------------|
| **중소기업** | **70%** | **30%** | 정부 7,000만 / 기업 3,000만 |
| **중견기업** | **50%** | **50%** | 정부 5,000만 / 기업 5,000만 |

**납부 절차:**
1. 협약 체결 → 가상계좌 발급
2. 가상계좌 발급 후 **2~5영업일 내 납부**
3. 분할납부: 최대 **3회**까지 가능 (협약 체결일 기준 6개월 이내, 사전 신청 필요)
4. 납부 완료 → 정부지원금 자동 매칭 → 바우처 발급

> 주의: 기한 내 미납 시 바우처 무효 또는 규모 축소. 부가세는 별도 납부 항목 (환급 가능)

### 3.4 선정 심사 기준

**서류 평가 (1차) 주요 항목:**
1. **기업의 수출역량** - 최근 수출 실적, 제품 경쟁력, 조직 구성
2. **사업계획의 구체성** - 예산 실현 가능성, 수행기관 연계 전략
3. **성과 목표 및 기대효과** - 수출 확대 가능성 평가

**가점 항목 (전체 배점 중 약 10~20점):**
- 벤처기업 인증
- 여성/장애인/사회적기업
- ESG 실천 기업
- 소재/부품/장비 전문기업
- 바우처 성실 정산 경험
- 스마트공장 구축 기업

**감점/탈락 사유:**
- 국세 또는 지방세 체납
- 휴/폐업 기업
- 과거 부정사용 이력
- 정산 미흡 이력
- 동일 대표자 중복 신청

### 3.5 조건 미달 시 보완 방법

| 미달 사항 | 보완 방법 |
|-----------|----------|
| 수출 실적 없음 | 내수기업 트랙으로 신청 (매출 10억+ 필요) |
| 사업자등록 1년 미만 | 등록 후 1년 경과까지 대기 |
| 자본잠식 | 증자 또는 재무구조 개선 |
| 구체적 수출 계획 부재 | 타겟 국가/바이어/제품 전략을 수립한 사업계획서 작성 |
| 수출 관련 이력 부재 | 해외 전시회 참가, 해외 거래 추진 실적 확보 |

## 4. 수행기관 등록

### 4.1 수행기관 현황

| 항목 | 수치 |
|------|------|
| 등록 수행기관 수 | **2,246개 기관** (2026.03 기준) |
| 등록 서비스 수 | **약 7,500여개** |
| 서비스 분야 | **16개 분야** (기존 14개 + 관세대응패키지 + 테크서비스) |
| 수행기관 선정 주기 | **수시 접수, 연간 6회 심사/선정** |

### 4.2 수행기관 분류

| 구분 | 설명 |
|------|------|
| **일반분야 수행기관** | 14개 서비스 분야 (컨설팅, 디자인, 통번역 등) |
| **특수분야 수행기관** | 해외인증, 의료기기, CBAM 등 전문 분야 |
| **테크서비스 수행기관** | AI/디지털 기술 기반 서비스 (2025년 신설) |

### 4.3 등록 자격 요건

**필수 요건:**
- 사업자등록 보유 법인 또는 개인사업자
- 해당 서비스 분야에서의 **전문성** 입증 가능
- 수출과 **직접 연관된** 서비스 제공 능력
- 수행 인력의 경력/자격 증명

**권장 요건:**
- **수출바우처 사업 경험 최소 2년 이상** 권장 (사업 이해도 및 정산 노하우)
- 재무적 안정성 (인력 및 재정 관리 능력)
- 유관 업종 실적 및 포트폴리오

**결격 사유 (등록 불가):**
- 서비스명과 실제 서비스 내용이 상이한 경우
- 직접적인 해외수출과 무관한 서비스 또는 국내 서비스만 제공
- 동일 서비스의 복수 등록
- 표준 가격 가이드 미준수
- 수행분야와 다른 명칭 사용
- 과거 연도가 포함된 서비스명 사용

### 4.4 등록 절차

```
1. 수출바우처 포털(exportvoucher.com) 회원가입/로그인
      |
      v
2. [수행기관] 메뉴 클릭 → 지원시스템 이동
      |
      v
3. [서비스 관리] → [서비스 등록/조회] 메뉴 접근
      |
      v
4. 서비스 정보 등록
   - 서비스 카테고리 선택
   - 서비스 명칭, 소개
   - 산업 분류, 타겟 국가
   - 기준 가격, 완료 기한
   - 포트폴리오
   - 환불 기준
   - 전문가 소개 (수행 인력 프로필)
      |
      v
5. 필수 서류 제출
   - 서비스 수행 계획서
   - 사업실적 증명서
   - 내부 수행 인력 개인별 증명서 (경력증명서, 자격증 등)
   - 특수관계기업 여부 확인서
   - 정보제공 동의서
   - 부패방지 서약서
      |
      v
6. 심사/평가
   - 전문성: 서비스 제공 적합성 + 수행 인력 경력
   - 수행 역량: 경험, 성공 사례, 수행 계획의 명확성/경쟁력
   - 안정성: 인력/재정 관리 능력
      |
      v
7. 선정 통보 (연 6회)
      |
      v
8. 수행기관 등록 완료 → 서비스 노출
```

### 4.5 수행기관 수수료/마진 구조

**핵심 포인트: 수행기관은 별도의 "수수료"를 받는 것이 아니라, 서비스 가격 자체가 매출이다.**

| 항목 | 내용 |
|------|------|
| 수익 구조 | 등록한 서비스의 가격 = 수행기관의 매출 |
| 가격 책정 | 수행기관이 자율적으로 설정 (단, **표준 가격 가이드 준수 필수**) |
| 정산 방식 | 서비스 완료 → 참여기업 확인 → 정산 요청 → 운영기관 승인 → 대금 지급 |
| 부가세 처리 | 참여기업이 수행기관에 부가세 별도 납부 |
| 마진 확보 방법 | 서비스 원가 대비 가격 설정 (단, 표준가 범위 내) |

**장점:**
- 정부 예산 기반의 **안정적 매출 확보** (정부가 70% 대납)
- 사실상 **미수금 리스크 제로** (정산 절차 완료 시 확정 지급)
- 수출 관련 **다수 중소기업 고객 확보** 가능
