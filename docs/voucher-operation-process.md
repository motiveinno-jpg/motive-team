# 바우처 수행기관 실무 프로세스 (2026-03-02)

> 모티브이노베이션 = 수출바우처 + 제조혁신바우처 수행기관 이미 등록 완료
> 핵심: 수혜기업 유치 → 서비스 수행 → 결과보고 → 정산 자동화

---

## 1. 수출바우처 수행기관 실무

### 전체 플로우
```
수혜기업이 exportvoucher.com에서 서비스 검색
→ 모티브(휘슬 AI) 서비스 선택
→ 포털 내 전자 계약 체결
→ 서비스 수행 (AI분석/서류생성/컨설팅)
→ 수행기관이 완료보고 + 결과물 업로드
→ 수혜기업이 서비스 완료 확인(승인)
→ 수행기관이 정산 요청 등록
→ 운영기관(KOTRA/중진공) 검토
→ 정부 지원금 수행기관에 지급
```

### 서비스 계약 필수 항목
- 계약 당사자: 수혜기업 + 수행기관(모티브)
- 서비스명: 등록된 서비스와 일치해야 함
- 수행 기간: 사업연도 내 완료 필수
- 공급가액: 표준 가격가이드 준수
- 부가가치세: 수혜기업이 별도 부담 (바우처 불가)
- 선금: 1,000만원 초과 + 2개월 이상인 경우, 50%까지 선금 가능
- 성과물/결과물 명세

### 결과보고서 필수 항목
1. 서비스 수행 개요 (기간, 대상 국가, 목표)
2. 과업 수행 내용 상세 기술
3. **실체적 과업물/성과물** (리포트, 서류, 디자인 시안 등 — 단순 미팅/자문만으로는 불인정)
4. 수출 성과 또는 기대 효과
5. 수행 인력 정보

### 정산 핵심 규칙
- 서비스 완료일로부터 **1개월 이내** 최초 정산 신청 필수
- 정산 보완 요청 시 **2주일 이내** 보완 (최대 3회)
- 사업연도 서비스 완료 마감: **11월 15일** (디자인: 10월 5일)
- 전자세금계산서: 수혜기업 완료 확인 시점에 발행
- 포털 외 계약은 정산 불인정

### 수혜기업 자부담 비율 + 지원 한도
| 단계 | 수출실적 | 지원 한도 | 정부 지원 |
|------|---------|---------|---------|
| 내수 | 1,000불 미만 | 3,000만원 | 50~70% |
| 초보 | 1,000불~10만불 | 3,000만원 | 50~70% |
| 유망 | 10만~100만불 | 4,500만원 | 50~70% |
| 성장 | 100만~500만불 | 7,000만원 | 50~70% |
| 강소 | 500만불 이상 | 1억원 | 50~70% |

## 2. 제조혁신바우처 수행기관 실무

### 전체 플로우 (3자 계약)
```
수요기업 ↔ 운영기관 2자 협약
→ 바우처 발급
→ 수요기업 + 수행기관(모티브) + 운영기관 3자 계약
→ 사업 수행
→ 완료보고 + 결과물 제출
→ 수요기업 완료 확인
→ 운영기관 검토
→ 정산
```

### 서비스 카테고리 (모티브 해당)
- **AI/DX 추진전략**: 디지털전환, AI 도입 전략
- **시스템 구축**: IT 시스템, 스마트공장
- **브랜드 지원**: BI, CI, 브랜드 전략
- **홍보물 제작**: 카탈로그, 영상, 웹사이트

### 자부담 비율
| 매출액 | 정부지원 | 자부담 |
|--------|---------|-------|
| 3억 이하 | 85~90% | 10~15% |
| 3~10억 | 75~80% | 20~25% |
| 10~50억 | 65~70% | 30~35% |
| 50~120억 | 45~50% | 50~55% |
| 120억 초과 | 40% | 60% |

- 최대 지원: 5,000만원/기업

---

## 3. 정산 시 필요 증빙 서류

### 수행기관(모티브) 제출
1. 사업결과보고서 (포털 양식)
2. 전자세금계산서
3. 서비스 수행 결과물 (실체적 과업물)
4. 교육 수료증 (교육 서비스인 경우)
5. 서비스 수행 계획 대비 실적 비교표

### 수혜기업 제출
1. 부가가치세 송금 확인증
2. 4대보험 가입자 명부 (최근 30일)
3. 서비스 완료 확인서 (포털 전자 승인)

## 4. 휘슬 AI 자동화 대상

### 계약 관리 자동화
- [ ] 바우처 계약 상태 추적 대시보드
- [ ] 선금 조건 자동 판단 (1,000만원 초과 + 2개월 이상)
- [ ] 계약 기간/마감일 알림 (11월 15일 카운트다운)

### 수행 관리 자동화
- [ ] 서비스 수행 진행률 대시보드
- [ ] 수혜기업 만족도 설문 자동 발송
- [ ] 과업별 수행 결과 자동 기록 (AI분석 횟수, 서류 생성 횟수)

### 결과보고서 자동 생성
- [ ] 포털 양식 기반 자동 채움
- [ ] 결과물 첨부 파일 관리 (PDF 리포트, 서류)
- [ ] 필수 항목 누락 검증
- [ ] 수출 성과 지표 자동 계산

### 정산 자동화
- [ ] 서비스 완료 확인 후 전자세금계산서 발행 트리거
- [ ] 정산 요청 등록 알림
- [ ] 부가세 납부 추적
- [ ] 정산 보완 요청 시 2주 타이머
- [ ] 증빙서류 체크리스트 자동 생성

## 5. 수혜기업이 휘슬 서비스를 바우처로 이용하는 시나리오

### 시나리오 A: AI 수출분석 서비스
```
수혜기업이 exportvoucher.com에서 "AI 수출분석" 검색
→ 모티브(휘슬 AI) 선택
→ 포털 내 계약 (공급가액: 500만원)
→ 휘슬 AI 계정 발급
→ 제품 URL 입력 → AI 분석 리포트 5건 생성
→ HS코드 분류 + FTA 시뮬레이션 + 타겟국 분석
→ 결과물: PDF 리포트 5건 + 요약 보고서
→ 결과보고서 자동 생성 → 포털 업로드
→ 수혜기업 승인 → 정산 (정부 70% + 기업 30%)
```

### 시나리오 B: 수출서류 자동생성 서비스
```
→ 포털 계약 (공급가액: 300만원)
→ 휘슬 AI로 PI/CI/PL/CO 자동 생성 (20건)
→ 결과물: 서류 원본 + 생성 이력
→ 결과보고서 → 정산
```

### 시나리오 C: 알리바바 입점 대행
```
→ 포털 계약 (공급가액: 1,000만원)
→ 알리바바 스토어 개설 + 상품 등록 + 운영
→ 결과물: 스토어 URL, 상품 페이지, 인콰이어리 대응 내역
→ 결과보고서 → 정산
```
