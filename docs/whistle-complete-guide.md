## 2. Whistle AI 서비스 개요

### 한 줄 소개
**"수출이 어렵지 않다. 제조사가 느끼는 벽을 AI와 플랫폼으로 허물어준다."**

### 서비스 정의
Whistle AI는 **AI 수출 인큐베이팅 플랫폼**이다. 단순한 수출 도구가 아니라, 수출 경험이 전혀 없는 제조사를 **0에서 수출 성공까지** 인큐베이팅하는 프로그램이다.

### 핵심 철학 (CEO 직접 정의, 2026-03-21)
> "말만 하는 서비스가 아니라, 실제 제조사와 바이어에게 지금까지 없던 우리만의 편리한 서비스를 만들어주고, 전세계 누구나 글로벌 수출제품으로 쉽게 연결해주는 것이 목표. 그것을 직원 1명의 급여보다 더 적은 금액으로."

### 핵심 가치 제안 (Value Proposition)
- **비용**: 직원 1명 월급(300~500만원)보다 저렴한 월 $99~$199로 수출 전 과정 자동화
- **시간**: 수출 준비 40시간 → 8시간으로 단축
- **접근성**: 상세페이지 URL 또는 사진 한 장만 있으면 60초 만에 AI 수출 전략 보고서 생성
- **원스톱**: 분석 → 인증 → 매칭 → 서류 → 결제 → 물류 → 보고까지 하나의 플랫폼에서 완결

### 6서비스 체인 (수출 여정 전체)
```
AI 수출분석 → 인증 대행 → 바이어 매칭 → 서류 자동생성 → 안전결제(에스크로) → 물류/통관
```

### 플랫폼 구조 (4자 연결)
```
제조사(Seller) ↔ 바이어(Buyer) ↔ 알리바바(Alibaba) ↔ 관리자(Admin)
```

## 3. 핵심 기능 (상세)

### 3-1. AI 수출 분석 엔진 (최핵심 기능)

**입력 방식 3가지:**
1. 제품 URL 입력 (쿠팡, 네이버, 아마존, 알리바바 등 15개 플랫폼 자동 크롤링)
2. 상세페이지/회사소개서/제품설명서 파일 업로드 (이미지, PDF)
3. 제품 정보 직접 입력

**분석 과정:**
- AI 3인 전문가 패널이 분석 (관세사 15년 / 물류전문가 20년 / 수출실무 12년 페르소나)
- URL 자동 크롤링 + 이미지 Vision AI + PDF 분석
- 실제 데이터 연동: UN Comtrade 무역통계, 관세청 API, 실시간 환율, World Bank 지표

**13개 분석 섹션 (60초 내 생성):**
1. 시장성 분석 (Market Potential)
2. HS코드 자동 분류 (AI + 관세청 API 교차검증)
3. 인증/규제 요구사항 (FDA, CE, 할랄, 약사법 등)
4. 경쟁사 분석
5. FOB 가격 산출
6. 타겟 시장 추천
7. 리스크 평가
8. 실행 로드맵 (단계별 일정)
9. 키워드 분석
10. 원가 시뮬레이션
11. 생산 가이드
12. 서류 체크리스트
13. 알리바바 적합도

**후속 액션:**
- PDF 보고서 다운로드
- 바이어 매칭 즉시 시작
- 수출 프로젝트 자동 생성
- 이메일로 보고서 공유

### 3-2. HS코드 자동분류

- AI가 제품 정보를 분석하여 HS코드 자동 분류
- 관세청 API와 교차검증으로 정확도 극대화
- 21개 FTA 협정, 59개국 관세율 자동 조회
- Edge Function `classify-hs` 실시간 처리

### 3-3. FTA 관세 최적화 시뮬레이션

- 21개 FTA 협정 데이터 탑재
- 59개국 관세율 자동 계산
- 5탭 구성: HS코드 / FTA 혜택 / 원가 분석 / 물류비 / 마진 시뮬레이션
- 국가별 관세 절감 금액 시각화

### 3-4. 제재 스크리닝 (Sanctions Screening)

- US CSL (Consolidated Screening List) 8개 리스트 실시간 조회
- SDN (Specially Designated Nationals)
- Entity List
- Denied Persons List 등
- 거래 상대방 자동 스크리닝

### 3-5. AI 바이어 매칭

- 제품 분석 결과를 기반으로 적합 바이어 AI 추천
- 매칭 점수 + 근거 시각화
- 9개 카테고리별 제조사/바이어 탐색:
  - K-Beauty, K-Food, Health Supplements, Electronics, Fashion, Organic, Home, Medical, Industrial
- 바이어 장터(Marketplace): 바이어가 요청 게시 → 제조사가 제안 입찰

### 3-6. 딜룸 (Deal Room) — 채팅 중심 거래 시스템

**채팅이 메인 UX** — 거래의 모든 과정이 채팅 안에서 진행

**9단계 딜 파이프라인:**
1. Inquiry (문의)
2. Matched (매칭)
3. Quotation (견적)
4. Sample (샘플)
5. Negotiation (협상)
6. Order (주문)
7. Production (생산)
8. Shipping (출하)
9. Completed (완료)

**딜룸 기능:**
- 실시간 1:1 채팅 (Supabase Realtime)
- Rich Card 메시지: 견적서, 서류, 결제, AI 가이드가 채팅 카드로 표시
- AI 자동 번역 (한/영/일/중 실시간)
- Action Chips: 주문관리, 물류추적, 원가계산, 서류생성, 샘플 관리, 선적 등록
- 전자서명: 서명 요청 → 알림 → 딜룸 배너 → 서명 수락

### 3-7. 샘플 결제 시스템

- 샘플 거래 전용 간소화 프로세스
- Stage 2 스킵 → 출하 확인 즉시 100% 캡처
- Stripe 결제 연동

### 3-8. 에스크로 안전결제 (3단계 자금 보호)

**Stripe manual capture 방식으로 구현**

| 단계 | 내용 | 자금 방출 |
|------|------|-----------|
| Stage 1: 주문 확인 | 바이어 결제 → 판매자 3영업일 내 수락 | 미수락 시 자동 취소 + 전액 환불 |
| Stage 2: 출하 확인 | 판매자 선적 서류 업로드 + 운송장 입력 | 50% 부분 캡처 (첫 정산) |
| Stage 3: 수취 확인 | 바이어 수취 확인 (14일 무응답 시 자동 확인) | 나머지 50% 캡처 (최종 정산) |

**환불 정책:**
| 시점 | 환불율 |
|------|--------|
| 주문 수락 전 | 100% |
| 출하 전 | 100% (판매자 동의 필요) |
| 출하 후, 수취 전 | 물류비 공제 후 환불 (협의) |
| 수취 확인 후 | 환불 불가 (하자 시 별도 분쟁 절차) |

**특수 케이스:**
- 바이어 수취 거부(하자): 사진/영상 증거 → 관리자 심사 → 전액/부분 환불 결정
- Enterprise 거래(대금 분할): 30/40/30 분할 캡처 (계약/출하/수취)
- 분쟁 제기 시: 관리자 수동 심사 (최대 30일)

**플랫폼 수수료:** 2.5% (Stripe 결제수수료 별도)

## 4. 타겟 고객

### 4-1. 한국 제조사/중소브랜드 (Primary Target)

**페인 포인트:**
- "수출은 어렵다"는 인식
- 내수시장 포화 → 해외 진출 필요성은 느끼나 방법을 모름
- 관세, 인증, 서류, 물류, 바이어 발굴 — 모든 것이 장벽
- 무역 전문인력 고용비용 부담 (월 300~500만원)

**세부 타겟:**
- 수출 경험 없는 내수 제조사 (첫 수출 도전)
- 수출 경험 있으나 비용/시간 절감이 필요한 기존 수출 제조사
- 네이버 스마트스토어 / 쿠팡 / 11번가 / 아마존 / Shopify / eBay / Etsy 등 플랫폼 셀러 중 자체 제조(직접 생산) 업체
- K-Beauty, K-Food, Health Supplements, Electronics, Fashion, Organic, Home, Medical, Industrial 9개 카테고리

**타겟 규모:**
- 한국 중소 제조업체 약 35만 개사
- 이 중 수출 미경험 업체 약 25만 개사

### 4-2. 글로벌 바이어 (Secondary Target)

**페인 포인트:**
- 한국 제품에 관심 있으나 소싱 채널이 제한적
- 한국어 장벽
- 품질 검증 어려움
- 소량 주문 거래처 확보 어려움

**세부 타겟:**
- 한국 제품 소싱을 원하는 해외 유통업체
- 아마존/eBay/Shopify 셀러 중 한국 제품 리셀러
- K-Beauty / K-Food 전문 수입업체
- OEM/ODM 발주를 원하는 해외 브랜드

**바이어 혜택:**
- 전체 무료 이용
- 제조사에게 줄 할인쿠폰 3장 보유 (Influencer 배지 획득 시 무제한)
- RFQ(요청) 등록 → 제조사가 제안 입찰
- 관세/FTA 자동 계산기

### 4-3. 정부 바우처 수혜기업

### 9-1. AI 인큐베이팅 (단순 도구 X, 프로그램 O)
