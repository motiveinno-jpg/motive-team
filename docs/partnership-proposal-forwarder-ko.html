Referral Program

화주 추천 프로그램

귀사의 기존 화주를 Whistle AI에 추천해 주세요.
화주가 유료 구독을 유지하는 한, 매달 추천 수수료가 지급됩니다.

🔗

전용 추천 코드

귀사 전용 추천 코드를 발급해 드립니다. 기존 화주에게 전달만 하시면 됩니다.

💵

지속적 수수료

추천 화주가 구독을 유지하는 한 매월 수수료가 지급됩니다. 해지 시 중단, 재구독 시 재개.

📦

추천 화주 = 내 화물

추천한 화주의 딜에 귀사가 자동으로 우선 배정됩니다. 화물을 뺏기지 않습니다.

📊

실시간 대시보드

추천 현황, 구독 상태, 수수료 내역을 실시간으로 확인할 수 있습니다.

S/I, CI, PL 등 필요 서류가 이미 생성된 상태로 전달됩니다. 확인만 하시면 됩니다.

🌏

알리바바 네트워크

Whistle AI는 알리바바 코리아와 전략적 파트너십을 진행 중입니다. 대량 화물 유입이 예상됩니다.

⭐

온라인 평판 구축

평점, 리뷰, 완료 건수가 프로필에 쌓입니다. 높은 평판 = 더 많은 화물 배정.

Onboarding

참여 절차

1. 제휴 신청

온라인 제휴 신청서를 작성해 주세요. (3분 소요)

2. 미팅 (30분)

플랫폼 시연 + 제휴 조건 상세 협의

3. 제휴 계약 체결

전자 계약서 서명 (초기 파트너 우대 조건 적용)

4. 프로필 등록

주요 항로, 운임 범위, 특수화물 가능 여부 등록 (10분 소요)

5. 화물 매칭 시작

프로필 승인 후 즉시 화물 배정 시작

함께 글로벌 물류를 연결합니다

가입비 무료, 월 구독료 무료. 화물이 성사될 때만 수수료가 발생합니다. 리스크 없이 시작하세요.
