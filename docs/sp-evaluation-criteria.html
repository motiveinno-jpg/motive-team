평가 항목 및 배점

SP의 서비스 품질은 4개 항목으로 평가되며, 각 항목은 5.0 만점으로 산정됩니다.

평가 항목배점 비중측정 방식기준
응답속도 30% 시스템 자동 측정 딜 배정 후 수락/거절까지의 시간. 12시간 이내 = 5.0, 24시간 = 4.0, 48시간 = 3.0
서류정확도 30% 시스템 자동 + 관리자 검토 제출 서류의 오류율. 무오류 = 5.0, 1회 수정 = 4.0, 2회 수정 = 3.0, 3회+ = 2.0
고객만족 20% 제조사 직접 평가 딜 완료 후 제조사가 1~5점으로 평가
완료율 20% 시스템 자동 측정 수락한 딜 중 정상 완료 비율. 100% = 5.0, 90% = 4.0, 80% = 3.0

평점 산정 방식

종합 평점 = (응답속도 x 0.3) + (서류정확도 x 0.3) + (고객만족 x 0.2) + (완료율 x 0.2)

만점: 5.0 | 딜 완료 시마다 갱신 | 최근 6개월 데이터 기준 이동 평균

평점은 딜 완료 시점에 제조사 평가(고객만족)와 시스템 자동 평가(응답속도, 서류정확도, 완료율)를 종합하여 산정됩니다. 최초 5건의 딜이 완료되기 전까지는 "신규" 표시로 등급이 보류됩니다.

등급 체계

등급평점 범위혜택/조치
Gold 4.5 ~ 5.0 우선 딜 배정, 수수료율 우대 (관세사 8%, 포워더 4%), 플랫폼 상단 노출, "우수 SP" 뱃지
Silver 4.0 ~ 4.4 일반 딜 배정, 표준 수수료율 (관세사 10%, 포워더 5%)
Bronze 3.5 ~ 3.9 일반 딜 배정, 분기별 개선 권고 메일 발송
Warning 3.0 ~ 3.4 신규 딜 배정 제한, 개선 계획서 제출 요구, 월 1회 모니터링
Suspend 3.0 미만 즉시 딜 배정 정지, 자격 정지 처리, 복귀 절차 필요

페널티 규칙

아래 사항 발생 시 즉시 평점이 차감됩니다.

위반 사항평점 차감비고
딜 기한 초과-1.0약정 기한 대비 지연 발생 시
서류 오류-0.5관세청/세관 반려 사유에 해당하는 오류
고객 불만 접수-0.5제조사의 공식 불만 접수 건
무응답 (48시간 초과)-1.0딜 배정 후 48시간 내 응답 없음
무단 딜 포기-2.0수락 후 사전 통보 없이 포기
3회 누적 페널티 발생 시 자격 정지 처리됩니다. 페널티는 6개월간 유지되며, 해당 기간 동안 무사고 시 초기화됩니다.

우수 SP 인센티브

Gold SP 전용 혜택

  • 딜 우선 배정: 동일 조건의 딜에서 Gold SP에게 먼저 배정
  • 수수료율 우대: 관세사 10% → 8%, 포워더 5% → 4%
  • 플랫폼 상단 노출: SP 목록에서 최상단 배치
  • "우수 SP" 뱃지: 프로필에 Gold 뱃지 표시
  • 분기별 우수 SP 시상: 플랫폼 뉴스레터 및 SNS 홍보

분기별 리뷰

매 분기 말(3월, 6월, 9월, 12월) 전체 SP에 대한 성과 리뷰를 실시합니다.

구분대상조치
상위 10%분기 최우수 SP인센티브 지급, 사례 공유
중위 80%일반 SP개별 성과 리포트 제공
하위 10%개선 필요 SP개선 요청 메일 + 1:1 컨설팅 제안
2분기 연속 하위 10%에 해당하는 SP는 계약 해지 검토 대상이 됩니다. 해지 전 반드시 서면 사전 통지(30일)를 진행합니다.

자격 정지 후 복귀 절차

1
대기 기간: 자격 정지일로부터 30일간 대기
2
재교육 이수: 플랫폼 사용법 및 서비스 품질 교육 재이수 (온라인)
3
개선 계획서 제출: 정지 사유에 대한 원인 분석 및 재발 방지 계획
4
테스트 딜 진행: 테스트 딜 1건 정상 완료 확인
5
복귀 승인: 관리자 최종 검토 후 Bronze 등급으로 복귀

복귀 후 3개월간 월별 모니터링을 실시하며, 동일 사유 재발 시 영구 자격 박탈 처리됩니다.
