## 1. 물류 & 배송 (Logistics & Shipping)

### 1-1. Freightos — 국제 운임 견적 API
- **제공**: 해상/항공/육상 실시간 운임 견적, 운임 지수(Freightos Baltic Index)
- **Whistle 가치**: 제조사가 수출 비용 시뮬레이터에서 실시간 운임을 즉시 확인. 견적서(PI) 자동 생성 시 운송비 자동 반영
- **연동 난이도**: Easy — REST API, JSON 응답, 무료 API 키 발급 가능
- **비용**: Free tier (시간당 100콜 제한) / 상용은 별도 협의
- **즉시 실행**: ✅ 개발자 포털에서 API 키 즉시 발급 → developers.freightos.com

### 1-2. Flexport — 디지털 포워딩 플랫폼
- **제공**: 국제 화물 운송, 통관, 실시간 추적, CO2 배출 추적, 문서 관리
- **Whistle 가치**: 수출 딜 진행 시 실시간 선적 추적을 Whistle 대시보드에 통합. 바이어에게 투명한 물류 가시성 제공
- **연동 난이도**: Medium — REST API(JSON), OAuth 인증, 문서화 양호
- **비용**: Per-use (화물 건당 과금)
- **즉시 실행**: ⚠️ 파트너 프로그램 신청 필요 → developers.flexport.com

### 1-3. Shippo — 멀티캐리어 배송 API
- **제공**: UPS/FedEx/DHL 등 85+ 캐리어 통합, 라벨 생성, 추적, 주소 검증
- **Whistle 가치**: 샘플 발송, 소량 수출 시 원클릭 배송 라벨 생성. 소형 제조사의 초기 수출 진입장벽 제거
- **연동 난이도**: Easy — REST API, 테스트 모드 무료
- **비용**: Free tier (월 30건) / Pay-as-you-go ($0.05/라벨)
- **즉시 실행**: ✅ 즉시 가입 가능

### 1-4. Tradlinx — 실시간 해상 가시성
- **제공**: 컨테이너 실시간 추적, 항구 혼잡도, ETA 예측
- **Whistle 가치**: 수출 딜 진행 중 선적 상태를 바이어/제조사 양쪽에 실시간 공유
- **연동 난이도**: Medium
- **비용**: Subscription 모델
- **즉시 실행**: ⚠️ 영업팀 컨택 필요 → tradlinx.com

### 1-5. CJ Logistics / Pantos Logistics (한국 대형 포워더)
- **제공**: 한국 출발 국제 물류, 통관, 창고, 풀필먼트
- **Whistle 가치**: 한국 제조사 특화 물류 서비스. "수출 물류까지 원스톱" 포지셔닝 강화
- **연동 난이도**: Hard — API 비공개, B2B 계약 필요
- **비용**: Per-use (화물 건당)
- **즉시 실행**: ⚠️ B2B 파트너십 미팅 필요

## 2. 무역 금융 (Trade Finance)

### 2-1. Tazapay — B2B 에스크로 결제
- **제공**: 국제 B2B 에스크로, 마일스톤 결제, 다통화 지원, API
- **Whistle 가치**: 딜 채팅에서 에스크로 결제 버튼 → 자금 안전 확보 → 바이어/셀러 신뢰 극대화. Whistle 에스크로 비전과 직결
- **연동 난이도**: Easy — REST API, Sandbox 무료, 문서화 우수
- **비용**: 거래 수수료 (약 1~3%)
- **즉시 실행**: ✅ Sandbox 즉시 가입 → docs.tazapay.com
- **★ 최우선 추천**: Whistle 에스크로 기능의 핵심 인프라

### 2-2. K-SURE (한국무역보험공사) — 수출보험
- **제공**: 수출보험, 수출신용보증, 환변동보험, 국가신용등급 API, K-Sight 빅데이터
- **Whistle 가치**: 바이어 신용 리스크 자동 평가. "이 바이어 거래해도 안전한가?" AI 분석에 K-SURE 데이터 활용
- **연동 난이도**: Medium — 공공데이터 Open API 일부 제공, 보험 가입은 오프라인
- **비용**: Free (Open API) / 보험료 별도
- **즉시 실행**: ✅ 공공데이터포털에서 국가위험지수/신용등급 API 즉시 활용 가능

### 2-3. Payoneer — 국제 B2B 결제
- **제공**: 크로스보더 B2B 결제, 다통화 수취, EC21 연동 실적 보유
- **Whistle 가치**: 바이어→제조사 대금 정산 인프라. Stripe(바이어 결제) + Payoneer(셀러 정산) 조합
- **연동 난이도**: Medium — B2B API 제공, 파트너 프로그램 존재
- **비용**: FX 마진 + 건당 수수료
- **즉시 실행**: ⚠️ 파트너 프로그램 신청 필요

### 2-4. Wise (TransferWise) Business API — 국제 송금
- **제공**: 40+ 통화 국제 송금, 중간환율 적용, KRW 수취 지원
- **Whistle 가치**: 소액 결제/샘플비/선수금 등 빠른 국제 송금. FX 수수료 최소화
- **연동 난이도**: Easy — REST API, Sandbox 제공
- **비용**: 건당 수수료 (0.4~1.5%)
- **즉시 실행**: ✅ Sandbox 즉시 가입 가능
- **주의**: KRW는 수취만 가능, 발송 불가

## 3. 통관 & 컴플라이언스 (Customs & Compliance)

### 3-1. Zonos — HS 코드 자동분류 + 관세/세금 계산
- **제공**: AI 기반 HS 코드 자동분류(100ms 이내), 200개국 관세/세금/수수료 계산, GraphQL API
- **Whistle 가치**: 수출 서류(CI/PL) 생성 시 HS 코드 자동 입력. 비용 시뮬레이터에 관세 자동 반영. 바이어에게 Landed Cost 즉시 제공
- **연동 난이도**: Easy — GraphQL API, 문서화 우수
- **비용**: Per-use (분류 건당 과금)
- **즉시 실행**: ✅ 개발자 계정 즉시 생성 → zonos.com/docs
- **★ 최우선 추천**: 수출 서류 자동화 + 비용 시뮬레이터 핵심

### 3-2. 관세청 UNI-PASS Open API — 통관 진행 조회
- **제공**: 수출이행내역, 화물통관진행정보, 수출신고 수리일/출항일, 선적중량
- **Whistle 가치**: 수출 딜 진행 상태를 관세청 데이터와 자동 동기화. "통관 완료" 자동 알림
- **연동 난이도**: Easy — 공공 Open API, 무료
- **비용**: Free
- **즉시 실행**: ✅ UNI-PASS 회원가입 후 API 키 발급 → unipass.customs.go.kr

### 3-3. 해외인증정보시스템 (certinfo.kr)
- **제공**: 국가별 인증제도, 인증마크 검색, 인증기관 정보
- **Whistle 가치**: AI 인증 가이드 엔진의 데이터 소스. "이 제품을 미국에 수출하려면 어떤 인증이 필요한가?" 자동 안내
- **연동 난이도**: Medium — 웹 크롤링 필요 (공식 API 없음)
- **비용**: Free (정부 운영)
- **즉시 실행**: ✅ 데이터 수집 즉시 가능

## 4. 인증 & 시험 (Certification & Testing)

### 4-1. SGS Korea — 글로벌 시험인증
- **제공**: CE, FDA, REACH, RoHS 등 전 세계 인증 시험. 190개국 네트워크
- **Whistle 가치**: 인증 필요 제조사 → SGS 바로 연결. 수출바우처(해외규격인증) 수행기관 연계
- **연동 난이도**: Hard — API 없음, B2B 파트너십
- **비용**: 인증 건당 (수십~수백만원)
- **즉시 실행**: ⚠️ 한국지사 파트너십 미팅 필요
- **전략**: 수출바우처 "해외규격인증" 분야 수행기관으로 SGS 연결 → 바우처 매출 발생

### 4-2. Intertek Korea — 시험·검사·인증
- **제공**: CB Scheme, FDA ASCA, HALAL, 안전인증. KTR/KTL과 파트너십 보유
- **Whistle 가치**: SGS와 동일 역할. 복수 인증기관 견적 비교 기능 제공 가능
- **연동 난이도**: Hard — API 없음
- **비용**: 인증 건당
- **즉시 실행**: ⚠️ 파트너십 미팅 필요

### 4-3. KTR (한국화학융합시험연구원) / KTL (한국산업기술시험원)
- **제공**: 한국 공인 시험기관, KC/KS 인증, 해외 인증 연계 시험
- **Whistle 가치**: 국내 시험 → 해외 인증 원스톱. Intertek와 MOU 체결로 CB Scheme 활용 가능
- **연동 난이도**: Medium — 일부 전자 접수 시스템 존재
- **비용**: 시험 건당 (정부 보조금 활용 가능)
- **즉시 실행**: ⚠️ 기관 협업 제안서 필요

### 4-4. 한국할랄인증원
- **제공**: HALAL 인증 대행, 이슬람 국가 수출 인증
- **Whistle 가치**: 식품/화장품 제조사의 중동·동남아 수출 시 필수 인증 연결
- **연동 난이도**: Easy — 단순 연결/소개 모델
- **비용**: 인증 건당
- **즉시 실행**: ✅ 파트너 등록 즉시 가능

## 10. 알리바바 바이어 데이터 접근 & Open Platform API

### 10-1. 알리바바 공개 바이어 데이터 접근 방법

| 방법 | 설명 | 실행 가능성 |
|------|------|------------|
| **RFQ 게시판 모니터링** | alibaba.com/trade/search → 바이어 RFQ(구매 요청) 공개 목록 열람. 카테고리/키워드 검색 가능 | ✅ 즉시 |
| **공개 프로필 수집** | 바이어 회사명, 국가, 구매 이력 일부 공개. 단, 연락처(이메일/전화)는 비공개 | ✅ 즉시 |
| **Alibaba Open API** | openapi.alibaba.com — 상품/주문/물류/정산 등 20+ 엔드포인트. JSON/XML 지원 | ⚠️ 셀러 계정 필요 |
| **3rd Party 스크래핑 API** | Oxylabs, Apify, ScrapingBee 등 — 상품/가격/리뷰 데이터 추출 | ✅ 유료 |
| **Trade Assurance 데이터** | 거래 보증 마크 보유 바이어/셀러 필터링 | ✅ 공개 |

**핵심 제약사항:**
- 바이어 **개인 연락처(이메일/전화)는 알리바바 내부에서만 접근 가능** → Trade Manager/앱 내 메시지만 허용
- 바이어 초대 최적 전략: **알리바바 RFQ에 공급자로 응답** → 대화 시작 → Whistle 초대
- 공개 데이터: 회사명, 국가, 구매 카테고리, 구매 빈도, Trade Assurance 여부

### 10-2. Alibaba Open Platform API 기능

| API 카테고리 | 주요 기능 | Whistle 활용 |
|-------------|----------|-------------|
| **Product API** | 상품 등록/수정/조회/삭제 | Whistle에서 상품 등록 → 알리바바 자동 동기화 |
| **Order API** | 주문 생성/조회/상태 변경 | 알리바바 주문 → Whistle 딜 자동 생성 |
| **Logistics API** | 배송 정보 등록/추적 | 선적 상태 양방향 동기화 |
| **Message API** | 바이어 메시지 송수신 | Whistle 채팅 ↔ 알리바바 메시지 통합 |
| **Image API** | 상품 이미지 관리 | 자동 이미지 동기화 |

**접근 조건:**
- 알리바바 Gold Supplier 계정 필요 (유료, 연 $1,999~$5,999)
- App Key 발급 → developer.alibaba.com
- Rate Limit 존재 (API별 상이)
- 바이어 개인정보 직접 접근 API는 **없음** (플랫폼 정책)

### 10-3. 알리바바 바이어 아웃리치 실행 전략

```
1단계: 알리바바 RFQ 모니터링 자동화
   → 카테고리별 새 RFQ 크롤링 → Whistle에 바이어 리드로 등록

2단계: Gold Supplier 계정으로 RFQ 응답
   → 알리바바 내에서 대화 시작 → Whistle 플랫폼 소개

3단계: Open API 연동
   → Whistle 상품 ↔ 알리바바 상품 동기화
   → 알리바바 주문 → Whistle 딜 자동 전환

4단계: 알리바바 총판대행 서비스
   → 한국 제조사 대신 알리바바 입점/운영 대행 (바우처 활용)
   → 대행 과정에서 바이어 데이터 자연 축적
```

## 핵심 조합 전략

```
[제조사 여정]
URL 입력 → Zonos(HS코드) + K-stat(트렌드) → AI 분석 리포트

알리바바 RFQ 모니터링 → Whistle 바이어 초대

→ Stripe/Tazapay(결제) → Flexport(물류) → 배송 완료

[바우처 연계]
수혜기업 유치 → Whistle 구독 = 바우처 결제
→ SGS/Intertek(인증) = 바우처 결제
→ 결과보고 자동 생성 = Whistle AI
```
