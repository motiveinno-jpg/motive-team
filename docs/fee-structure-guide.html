4. SP 추천 수수료 (레퍼럴)

SP가 신규 제조사를 휘슬 플랫폼에 추천하여 유료 구독 전환 시, 해당 구독료의 일정 비율을 지속적으로 수령합니다.

SP 등급 추천 수수료율 기준 지급 방식
Silver 8% 기본 등급 구독료 기반, 매월 지속 지급
Gold 12% 추천 5건 이상 구독료 기반, 매월 지속 지급
Platinum 15% 추천 15건 이상 구독료 기반, 매월 지속 지급
Diamond 20% 추천 30건 이상 구독료 기반, 매월 지속 지급

추천 수수료 지급 조건

  • 추천받은 제조사가 유료 구독(Starter 이상)을 유지하는 동안 지속 지급
  • 구독 해지 시 해당 건의 추천 수수료 지급 중단
  • 등급은 누적 유효 추천 건수 기준으로 자동 승급
  • 부정 추천(자기 추천, 허위 계정 등) 적발 시 자격 박탈

6. 부가세 안내

부가가치세 별도

  • 본 안내서의 모든 수수료는 부가가치세(VAT) 별도 금액입니다.
  • 한국 사업자: 부가세 10% 별도 부과, 세금계산서 발행
  • 해외 사업자: 해당 국가의 세법에 따라 처리
  • 구독료(USD)는 Stripe를 통해 청구되며, 환율은 결제 시점 기준 적용

7. 수수료 계산 예시

예시: $10,000 수출 거래

거래 금액 $10,000.00
Stripe 결제 수수료 (~3.4%, 바이어 부담) -$340.00
에스크로 수수료 (2.5%, 판매자 부담) -$250.00
판매자 순 수령액 $9,750.00

관세사 수임료 $500 별도 시: 수수료 10% = $50 차감, 관세사 수령 $450

포워더 운임 $2,000 별도 시: 수수료 5% = $100 차감, 포워더 수령 $1,900
