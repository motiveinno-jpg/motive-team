1. 결제 흐름 개요

Whistle AI 플랫폼의 모든 거래 대금은 플랫폼을 경유하여 결제되며, 수수료 차감 후 판매자 및 서비스 제공자(SP)에게 정산됩니다.

1
제조사 / 바이어
Stripe를 통해 결제 진행
↓
2
Whistle AI 에스크로
결제 대금 플랫폼 보관 (Stripe Authorization Hold)
↓
3
수수료 차감
에스크로 2.5% + Stripe 수수료 ~3.4% 공제
↓
4
SP / 판매자 정산
Stripe Connect를 통해 자동 송금

2. 에스크로 3단계

거래 대금은 3단계에 걸쳐 캡처(확정)됩니다. 각 단계 미확인 시 자동 확인 정책이 적용됩니다.

S1
Stage 1 — 주문 확인
바이어 결제 완료 후, 판매자가 3영업일 이내 주문을 수락합니다.
미수락 시 자동 취소 및 전액 환불 처리됩니다.
↓
S2
Stage 2 — 출하 확인 (50% 캡처)
판매자가 출하를 확인하면 거래 대금의 50%가 캡처됩니다.
운송장 번호 또는 B/L 업로드가 필요합니다.
↓
S3
Stage 3 — 수취 확인 (나머지 50% 캡처)
바이어가 물품 수취를 확인하면 나머지 50%가 캡처됩니다.
14일 무응답 시 자동 수취 확인 처리됩니다.

3. SP(서비스 제공자) 정산

3-1. 관세사 정산

트리거: 수출신고필증 업로드 완료

프로세스: 수출신고필증 업로드 → 시스템 자동 검증 → 수임료 자동 캡처 → 수수료(10%) 차감 → SP 정산

수수료: 수임료의 10%

3-2. 포워더 정산

트리거: B/L(선하증권) 또는 AWB(항공화물운송장) 업로드 완료

프로세스: B/L 또는 AWB 업로드 → 시스템 자동 검증 → 운임 자동 캡처 → 수수료(5%) 차감 → SP 정산

수수료: 운임의 5%

4. 정산 주기

구분 정산 시점 방법
에스크로 정산 (판매자) 캡처 확정 후 7영업일 이내 Stripe Connect 자동 송금
관세사 정산 수출신고필증 업로드 후 7영업일 이내 Stripe Connect 자동 송금
포워더 정산 B/L 또는 AWB 업로드 후 7영업일 이내 Stripe Connect 자동 송금

5. 수수료 공제 요약

항목 수수료율 부담 주체
에스크로 수수료 2.5% 판매자
관세사 중개 수수료 수임료의 10% 관세사(SP)
포워더 중개 수수료 운임의 5% 포워더(SP)
Stripe 결제 수수료 ~3.4% 바이어

6. 정산 명세서

발행 주기: 월 1회 자동 발행 (매월 5일, 전월 거래분)

확인 방법: SP 대시보드 > 정산 내역 메뉴에서 조회 및 PDF 다운로드

포함 내역: 거래별 금액, 수수료 내역, 순정산 금액, 송금 일시, Stripe Transfer ID

세금계산서: 정산 명세서 기반으로 별도 발행 (부가세 별도)

7. 전체 정산 흐름도

B
바이어 결제 — Stripe Checkout으로 카드 결제
↓
H
Authorization Hold — 결제 대금 플랫폼 보관
↓
C
단계별 캡처 — S1 주문확인 → S2 출하(50%) → S3 수취(50%)
↓
D
수수료 차감 — 에스크로 2.5% + SP 수수료 공제
↓
T
자동 송금 — Stripe Connect → 판매자/SP 계좌
↓
R
명세서 발행 — 월별 정산 명세서 자동 생성

유의사항

  • Stripe Authorization Hold의 유효 기간은 최대 7일입니다. 기한 내 캡처되지 않으면 재결제가 필요합니다.
  • 정산 금액이 $1 미만인 경우 다음 정산 주기로 이월됩니다.
  • 해외 송금 시 수취 은행의 수수료가 별도로 발생할 수 있습니다.
  • 분쟁 발생 시 정산이 보류되며, 분쟁 해결 후 처리됩니다.
