1. 에스크로 환불 규정

에스크로 거래의 환불은 거래 진행 단계에 따라 차등 적용됩니다.

거래 단계 환불 비율 조건
주문 수락 전 100% 환불 무조건 전액 환불. 바이어 요청 즉시 처리
주문 수락 후 ~ 출하 전 100% 환불 판매자 동의 필요. 판매자 거부 시 분쟁 처리
출하 후 ~ 수취 확인 전 협의 환불 물류비 공제 후 환불. 양측 협의 또는 관리자 중재
수취 확인 후 환불 불가 하자 발견 시 분쟁 제기를 통해 별도 심사

에스크로 환불 유의사항

  • 14일 자동 수취 확인 이후에는 환불이 불가하며, 분쟁 절차를 통해서만 처리됩니다.
  • 출하 후 환불 시, 반품 물류비는 환불 요청자가 부담합니다.
  • 판매자 동의 환불의 경우, 판매자에게 48시간의 응답 기한이 주어집니다.

2. SP 서비스 환불 규정

SP(관세사, 포워더 등)가 제공하는 서비스에 대한 환불은 아래 기준을 따릅니다.

2-1. SP 미수락 / 미착수

상황 환불 비율 추가 조치
SP 48시간 내 미수락 100% 환불 자동 재매칭 진행
SP 수락 후 72시간 미착수 100% 환불 SP에게 경고 부여

2-2. 제조사(의뢰인) 취소

상황 환불 비율 SP 보상
SP 착수 전 취소 100% 환불 SP 보상 없음
SP 착수 후 취소 70% 환불 30%를 SP에게 보상

2-3. 딜 취소 (거래 전체 취소)

상황 환불 비율 SP 보상
SP 착수 전 딜 취소 100% 환불 SP 보상 없음
SP 착수 후 딜 취소 70% 환불 30%를 SP에게 보상

2-4. SP 기한 초과

조건: SP가 약정 기한을 초과하여 서비스 미완료

환불: 100% 전액 환불

추가 조치: SP에게 페널티 부여 (반복 시 자격 정지)

3. 환불 처리 기간

결제 수단 환불 처리 기간 비고
카드 결제 (Stripe) 3~5 영업일 카드사 처리 기간에 따라 상이
계좌 이체 5~7 영업일 은행 영업일 기준

4. 분쟁 시 처리

관리자 수동 심사: 환불 관련 분쟁 발생 시, 관리자가 양측 증거를 검토하여 최종 결정합니다.

심사 기간: 최대 30일 (복잡한 사안의 경우)

심사 기준: 거래 기록, 배송 증빙, 커뮤니케이션 내역, 상품 상태 증거

결정 유형: 전액 환불 / 부분 환불 / 환불 불가 / 양측 합의 유도

5. 환불 신청 방법

1
앱 내 신청
거래 상세 페이지에서 "환불 요청" 버튼 클릭 → 환불 사유 선택 → 증빙 자료 첨부
↓
2
이메일 신청
cs@whistle-ai.com으로 주문번호, 환불 사유, 증빙 자료를 첨부하여 발송
↓
3
접수 확인
접수 후 1영업일 이내 접수 확인 알림 발송
↓
4
처리 완료
환불 승인 시 결제 수단별 처리 기간에 따라 환불 진행

환불 불가 사유

  • 수취 확인 완료 후 단순 변심에 의한 환불 요청
  • 바이어의 주문 사양과 일치하는 상품에 대한 환불 요청
  • 환불 요청 시 증빙 자료 미제출 (분쟁 건)
  • 플랫폼 이용약관을 위반한 거래
