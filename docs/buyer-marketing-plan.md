## 3. 바이어 온보딩 자동화 (4단계)

1. **가입 (30초)**: 이메일/구글/LinkedIn SSO → 회사명, 국가, 관심 카테고리 3개
2. **AI 첫 경험 (2분)**: 인터랙티브 투어 → 챗봇 "뭘 찾으세요?" → 즉시 3곳 매칭
3. **프로필 완성 (24시간 내)**: 이메일 유도 → 사업자증 업로드 → Verified 배지
4. **첫 인콰이어리 (48시간 내)**: 매칭 제조사에 1클릭 문의 템플릿
