# Whistle AI Demo Scenario — 거래 실행 OS

> 대상: 고객(제조사 대표) / 파트너(알리바바) / 투자자
> 준비: whistle.html 로그인 상태, 딜 1건 이상 존재

---

## 휘슬 차별점 5줄

1. **유일한 풀사이클** — 바이어 발굴 → 매칭 → 견적 → 협상 → 주문 → 생산 → 출하 → 완료, 전 과정 하나의 화면
2. **AI가 직접 실행** — 분석 리포트가 아니라, "다음에 뭐 해?" 물으면 버튼 하나로 다음 단계 실행
3. **5개 자연어 명령** — 다음 단계 / 정체 거래 / 바이어 신뢰도 / 가격 비교 / 도움말, 한국어로 질문
4. **바우처 직접 연계** — 수출바우처 수행기관으로서 정부 지원금 직접 활용 가능
5. **기존 도구의 1/40 가격** — 월 9,900원부터, Flexport/Zonos 대비 압도적 가격 경쟁력

---

## 질문별 기대 응답 요약

### 1. "뭐 할 수 있어?" (help)
- **카드 타입**: empty (안내)
- **내용**: 4개 명령 카드
  - 다음 단계 확인: "이 딜 다음에 뭐 해야 해?" → [실행]
  - 정체 거래 리포트: "뭐가 막혀있어?" → [실행]
  - 바이어 신뢰도 확인: "이 바이어 믿을 만해?" → [실행]
  - 가격 비교 분석: "이 제품 가격 어때?" → [실행]
- **후속 칩**: 없음

### 2. "뭐가 막혀있어?" (deal.stale_list)
- **카드 타입**: list (amber 보더)
- **내용**: 긴급/정체 거래 스크롤 목록
  - 각 항목: 아이콘 + 거래명 + 경과일 + LOW TRUST 배지 + [열기] 버튼
- **후속 칩**: [📋 다음 단계 확인] [🔍 바이어 신뢰도]

### 3. "다음 단계 뭐야?" (deal.next_step)
- **카드 타입**: status 또는 alert (넛지 있으면)
- **내용**:
  - 요약: "ABC Trading 거래는 현재 [견적] 단계입니다. 5일째 진행 중."
  - 단계 진행 바: ●──●──◉──○──○
  - 근거 칩: 신뢰 B | 거래가 $8.50 | 경과 5일
  - 리스크: (있으면) 저신뢰 경고, 정체 경고
  - 제안: "다음 단계: [샘플] — 샘플 발송 준비를 시작하세요"
  - 액션: [📦 샘플 진행] [💬 채팅]
- **후속 칩**: [🔍 바이어 신뢰도] [📊 가격 비교]

### 4. "이 바이어 믿을 만해?" (trust.check)
- **카드 타입**: status 또는 alert (D등급이면)
- **내용**:
  - 종합 등급: A~D + 점수
  - 5축 바 차트: 응답속도 / 결제이력 / 거래완료 / 서류 / 프로필
  - 등급별 조언: A "안심 진행" / B "정상 진행" / C "보수적 설정" / D "선결제 또는 에스크로"
- **후속 칩**: [📋 정체 거래 확인] [💰 가격 비교]

### 5. "이 제품 가격 어때?" (price.check)
- **카드 타입**: status 또는 alert (이상치 있으면)
- **내용**:
  - 품목별 가격 포지션 바 (시장 범위 대비 위치)
  - 이상치 감지: 📈 높음 / 📉 낮음 + 편차 %
  - 카테고리별 시세 범위
- **후속 칩**: [🔍 바이어 신뢰도] [📋 정체 거래 확인]
