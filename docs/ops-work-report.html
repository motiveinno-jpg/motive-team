어드민 운영가이드 메뉴 통합

admin.html 사이드바에 📚 운영 가이드 메뉴를 추가했습니다.
SP 관리 메뉴 바로 아래 위치하며, 15개 문서를 3개 카테고리로 분류하여 표시합니다.
각 문서를 클릭하면 새 탭에서 열립니다.

Admin > 사이드바 > 📚 운영 가이드
추가된 항목:
- 사이드바 메뉴: opsGuide (SP관리 아래, 제재 위)
- 라우팅: renderPage() → pgOpsGuide()
- 번역: 한/영 15개 문서명 + UI 텍스트
- 페이지 타이틀: PL.opsGuide

문서별 주요 내용
1. 관세사 제휴 계약서
업무 범위(HS코드·FTA·수출신고·원산지증명), 수수료 10%, 수출신고필증 업로드 시 자동정산, 환불 규칙 7가지, 계약 1년 자동연장, 서울중앙지법 관할
2. 포워더 제휴 계약서
운임 견적·부킹·B/L발급·창고관리, 수수료 5%, B/L·AWB 업로드 시 자동정산, 동일 환불 규칙, 동일 법적 조항
3. SP 이용약관
가입 자격, SP 의무(48h수락·72h착수·정확성), 금지행위(플랫폼 우회·가격담합), 페널티(기한초과=-1점, 3회=자격정지), 계정해지
4. 개인정보처리동의서
수집 항목(사업자번호·계좌·자격증), 보유 5년, Stripe 제3자 제공, 동의 거부권, 파기 절차
5. NDA 비밀유지계약
비밀정보 범위, 계약기간+3년 유지, 위약금 1천만원, 반환의무, 예외 규정
6. 정산 프로세스 가이드
에스크로 3단계 플로우(주문확인→출하50%→수취50%), SP 정산 트리거, 7영업일 Stripe Connect, 수수료 공제 구조
7. 환불 규정
에스크로 4단계 환불율, SP 7가지 환불 시나리오, 처리 기간(카드 3-5일/계좌 5-7일), 분쟁 시 30일
8. 수수료 체계 안내서
구독료 4단계, 에스크로 2.5%, 관세사 10%, 포워더 5%, Stripe 3.4%, SP 추천 수수료 티어(8~20%)
9. 분쟁 처리 규정
6가지 분쟁 유형, 접수→소명→심사→결정 절차, 증거 기준, 이의신청 1회, 서울중앙지법
10. 자동정산 오류 매뉴얼
5가지 오류 유형별 원인→감지→대응→에스컬레이션, 웹훅 재시도 로직, 수동 캡처 절차
11. SP 온보딩 가이드
7단계: 가입신청→심사→계약→프로필→정산계좌→교육→첫딜, 소요 5-7영업일
12. SP 평가/관리 기준
4항목(응답속도30%+정확도30%+만족20%+완료율20%), 5등급, 페널티, 우수 인센티브, 분기 리뷰
13. 딜 단계별 운영 매뉴얼
8단계: 바이어매칭→협상→관세사매칭→포워더매칭→생산검사→결제→선적통관→도착정산, 각 단계별 역할·서류·트리거
14. 카테고리별 서류 체크리스트
8개 카테고리(식품/화장품/전자/의류/건강기능/의료기기/생활용품/산업재), 발급기관·필수여부·유효기간
15. 긴급 상황 대응 매뉴얼
5가지 시나리오(서비스장애/결제사고/데이터유출/SP사고/법적분쟁), 15분 초동대응, 긴급 연락처
