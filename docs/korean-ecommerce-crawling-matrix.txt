─────────────────────────────────────────
✅ 추천 순서:

  2. Google Trends

UseCase 3: 경쟁사 분석 (Competitor Analysis)
──────────────────────────────────────────────
✅ 추천 순서:
  1. 공식 API (각 플랫폼)
  2. 수동 모니터링 (스크린샷)
  3. 데이터 제공업체
❌ 피하기: 브라우저 자동화 (탐지 위험)

UseCase 4: 상품 검색 기능 (Product Search)
───────────────────────────────────────────
✅ 추천 순서:
  1. API 통합 (Shopify, Cafe24, 11st)
  2. 직접 링크 제공 (각 플랫폼 검색)
❌ 피하기: 크롤링 + 재배포

UseCase 5: 마케팅/추천 (Marketing/Recommendation)
────────────────────────────────────────────────────
✅ 추천 순서:
  1. 판매자 계정 + API (해당하면)
  2. Affiliate Program (각 플랫폼)
  3. 광고 API (Google Shopping, etc.)
❌ 피하기: 스크래핑 + 무단 재사용
