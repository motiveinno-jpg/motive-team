## 1. 핵심 미션

**"수출이 어렵지 않다. 휘슬이 AI와 플랫폼으로 벽을 허문다."**

### 2.3 핵심 플로우 (제조사 관점)

```
[1단계] 제조사 진입
  ├── URL 입력 또는 자료 업로드
  └── AI 수출 분석 리포트 생성 (세계 최고 수준)
       ├── 타겟 국가/시장 분석
       ├── HS코드 자동 분류 (2→4→6→10자리)
       ├── FTA/관세 시뮬레이션
       ├── 경쟁사 분석
       ├── 필요 인증/서류 목록
       ├── 초기 물량 추천
       ├── 총 비용 예측
       ├── 소요 시간 예측
       └── 전문가 연결 추천

[2단계] 수출 준비
  ├── 수출바우처 자격 진단 → 수혜기업 만들어줌 → 휘슬 락인
  ├── PI/CI/PL 서류 자동 생성
  ├── HS코드 기반 도착단가 계산
  ├── 필요 인증 가이드 + 대행 연결
  └── 제품 카드 생성 → 바이어 검색 공개

[3단계] 바이어 매칭
  ├── 자체 바이어 네트워크 매칭
  ├── 알리바바 입점 대행 (총판권)
  ├── 바이어 RFQ 자동 알림
  └── AI 자동 번역 채팅

[4단계] 거래 진행
  ├── 견적 → PI → 계약금(30%) → 생산 → 검수 → 잔금(70%) → 출하
  ├── 에스크로 안전결제 (Stripe)
  ├── 물류 추적
  └── 수출 서류 자동 갱신

[5단계] 성과 관리
  ├── 수출바우처 결과보고 자동화
  ├── 거래 실적 대시보드
  └── 재거래 / 추가 바이어 매칭
```

## 3. 수익구조 5축 상세 설계

### 3.1 축 1: 수출바우처 수행기관

| 항목 | 내용 |
|---|---|
| **등록 카테고리** | 조사/일반 컨설팅 + 서류대행/현지등록 + 테크서비스 |
| **서비스 1** | AI 수출시장 분석 리포트 (300만~1,000만원/건) |
| **서비스 2** | AI 수출서류 자동생성 (50만~200만원/건) |
| **서비스 3** | HS코드 분류 + FTA 활용 컨설팅 |
| **정부 지원** | 70% (제조사 실부담 30%) |
| **예상 매출 Y1** | 20건 × 500만원 = 1억원 |
| **예상 매출 Y3** | 100건 × 700만원 = 7억원 |

**자격 부족 제조사 → 수혜기업 만들기 프로세스:**
```
진단: 사업자 1년↑? 매출 10억↑? 자본잠식 X? 수출계획 有?
  ├── 전부 OK → 바로 바우처 신청 지원
  └── 미달 항목 발견
       ├── 매출 부족 → 국내 매출 확대 전략 컨설팅
       ├── 수출계획 부재 → 휘슬 AI 리포트로 수출계획서 자동 생성
       ├── 자본잠식 → 재무구조 개선 가이드
       └── 사업자 1년 미만 → 대기 + 그간 준비사항 가이드

  ** 모든 보완 비용은 휘슬 서비스로 사용 (락인) **
```

**결과보고 자동화:**
```
바우처 서비스 수행 완료
  → 사업결과보고서 자동 생성 (AI)
     ├── 서비스 수행 내역 (분석 리포트/서류 생성 이력)
     ├── 수출 성과 지표 (바이어 접촉 수, 견적 수, 계약 수)
     ├── 전자세금계산서 연동
     └── 결과물 증빙 자동 첨부
  → 수행기관 정산 요청
  → 운영기관 승인 → 대금 수취
```

### 3.2 축 2: 알리바바 총판/대행

| 항목 | 내용 |
|---|---|
| **파트너 유형** | Channel Partner + VAS Partner 겸업 |
| **GGS 판매 커미션** | 멤버십 10~20% (예: Standard $4,999 → 커미션 ~$500~1,000) |
| **운영대행 BASIC** | $2,500 (3개월, 200개 상품등록) |
| **운영대행 PLUS** | $4,500 (3개월, 300개 상품등록) |
| **월정 운영관리** | $300~1,000/월 |
| **광고 관리** | 광고비의 15~20% |
| **성과 수수료** | 거래액의 3~5% (협의) |

**휘슬 AI 자동화 이점:**
- 제품 등록: AI가 영문 상세페이지 자동 생성 → 등록 시간 80% 절감
- 인콰이어리 응답: AI 1차 자동 응답 → 24시간 응답률 100% 달성
- RFQ 대응: AI 초안 + 사람 검토 → 대응 속도 3배
- 성과 리포트: 실시간 자동 대시보드

### 3.3 축 3: 서류 건당 과금

| 서류 | 가격 | 비고 |
|---|---|---|
| PI (Proforma Invoice) | 5,000원 | 무료 1건 |

### 3.4 축 4: 에스크로 수수료

| 거래 금액 | 수수료 | 비고 |
|---|---|---|
| $1,000 이하 | 5% | 소액 거래 |
| $1,001~$10,000 | 3% | 중소 거래 |
| $10,001~$50,000 | 2% | 알리바바 수준 |
| $50,001 이상 | 1.5% | 대형 거래 |

**기술 구현:** Stripe Connect (Separate Charges & Transfers)

### 3.5 축 5: 구독 (SaaS)

| 플랜 | 가격 | 포함 |
|---|---|---|
| Free | 0원 | AI 분석 1건, 서류 1건 |
| Starter | 99,000원/월 | AI 분석 5건, 서류 무제한, 바이어 매칭 3건 |
| Pro | 299,000원/월 | AI 분석 무제한, 서류 무제한, 바이어 매칭 무제한, 알리바바 연동 |
| Enterprise | 협의 | 전담 매니저, 커스텀 |

## 4. 바이어 유치 전략

### 4.1 한국 트렌드 상품 미끼 (기존 아이디어 고도화)

```
네이버 쇼핑 트렌드 API → 실시간 인기 상품 수집
  → 카테고리별 정리 (K-Beauty, K-Food, K-Fashion 등)
  → 해외 바이어에게 "한국에서 지금 핫한 상품" 푸시
  → "이 제품 소싱 원해? 매칭해줄게" CTA
  → 바이어 가입 유도
```

### 4.2 추가 바이어 유치 채널

| 채널 | 방법 | 효과 |
|---|---|---|
| SEO/콘텐츠 | "Korean [product] supplier" 키워드 타겟 블로그 | 장기 유입 |
| Google Ads | "Korean cosmetics manufacturer" 등 B2B 키워드 | 즉시 유입 |
| LinkedIn | 타겟 바이어 국가별 DM + 컨텐츠 마케팅 | 고품질 리드 |

### 4.3 바이어 락인 전략

- 안전결제(에스크로) → "휘슬에서 결제하면 안전"

- 제조사 품질 인증 → "검증된 한국 제조사만"

## 5. OEM/ODM 플랫폼 확장 (Phase 3)

```
바이어가 제품 제작 의뢰 등록
  ├── 카테고리 선택 (화장품, 식품, 패션 등)
  ├── 요구 사양 입력 (성분, 용량, 디자인 등)
  ├── 예산 범위 설정
  └── 납기 희망일

→ AI가 적합 제조공장 매칭
  ├── 공장 DB에서 카테고리/역량/가격대 매칭
  ├── 공장 신뢰도 스코어 (리뷰, 거래이력, 인증)
  └── TOP 3 공장 추천

→ 견적 / 샘플 / 계약 / 생산 / QC / 출하
  └── 전 과정 에스크로 결제 + 물류 추적
```
