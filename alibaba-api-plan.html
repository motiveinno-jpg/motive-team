서비스 개요

Whistle AI란?

한국 중소 제조사를 위한 수출 통합 관리 플랫폼입니다. HS코드 분류, FTA 원산지 판정, 바이어 매칭, 수출 서류 생성, 물류 관리까지 수출의 전 과정을 AI 기반으로 자동화합니다.

알리바바 대행 서비스 유형

모티브이노베이션은 두 가지 방식으로 한국 제조사의 알리바바 입점 및 운영을 지원합니다.

총판운영

Distributor 모델

모티브 자체 계정 운영

모티브 계정으로 상품 등록, 관리, 판매를 대행합니다. 다수 제조사의 상품을 하나의 스토어에서 통합 운영하여 효율을 극대화합니다.

계정대행

Managed 모델

고객사 자체 계정 운영

고객사 명의의 알리바바 계정을 모티브가 운영 대행합니다. 브랜드 독립성을 유지하면서 전문 운영 역량을 제공합니다.

목표

휘슬 플랫폼 안에서 알리바바 운영 현황을 실시간으로 확인하고, 데이터 기반 의사결정을 지원하는 것이 핵심 목표입니다.

알리바바 탭 UI 설계
📊

대시보드

조회수, 문의(RFQ) 수, 주문 수, 전환율 추이를 한눈에 파악. 기간별 비교 및 트렌드 차트 제공.

📦

상품 관리

등록 상품 목록, 상태(활성/비활성), 개별 상품 조회수 및 문의 수 확인. 상품 수정/상태 변경.

📨

문의 / 주문

들어온 RFQ 목록, 바이어 정보, 주문 상태, 금액을 실시간 확인. 읽음/미읽음 처리.

🏷

서비스 유형 표시

각 제조사별 총판 대행 뱃지로 서비스 유형 구분.

5
기술 연동 로드맵
P1
즉시 실행 가능

Phase 1 — 기본 데이터 수집

  • 브라우저 자동화(Playwright)로 스토어 기본 데이터 수집
  • 상품 목록, 조회수, 기본 통계 정보 동기화
  • 휘슬 대시보드에 기본 현황 표시
P2
API 확보 후

Phase 2 — 공식 API 연동

  • 알리바바 공식 API를 통한 실시간 데이터 동기화
  • 문의(RFQ), 주문, 메시지 양방향 연동
  • Webhook 기반 이벤트 알림 시스템 구축
P3
고도화 단계

Phase 3 — AI 기반 최적화

  • AI 기반 상품 설명 최적화 및 자동 번역 (다국어)
  • 시장 데이터 분석을 통한 가격 추천 시스템
  • 바이어 행동 분석 및 맞춤 마케팅 자동화
