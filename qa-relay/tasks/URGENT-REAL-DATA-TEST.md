## ★★★ 공통 필수 규칙

1. **"확인됨" = 스크린샷 + 실제 데이터 결과**. UI가 보인다고 "확인"이 아님
2. **AI 분석은 실제 URL 넣고 결과 전체 확인**. 빈 폼 클릭은 테스트가 아님
3. **서류는 실제 생성 + PDF 다운로드 + 내용 정확성 확인**
4. **비용 계산은 실제 숫자 + 결과를 직접 계산해서 검증**
5. **한 라운드 끝나면 다른 제품으로 다시 시작. 절대 멈추지 마.**
6. **사용자에게 질문하지 마. 자율 진행.**
7. **문제 발견 시 즉시 qa-relay/bugs/에 기록**
