## 핵심 발견 TOP 10

1. **글로벌 수출 올인원 SaaS = 세계에 없다** → 블루오션 확정
2. **무료 API 5개로 80% 뼈대** → CSL + 환율 + Vizion + emissions.dev + HSK
3. **FTA 절감액 = 최강 세일즈 훅** → 한국 중소기업 88% 원산지 판정 어려움
4. **블록체인 L/C = 컨소시엄 모델 실패 확정** → Stripe + Wise 이중 구조
5. **Stripe $100K 거래 시 $5,400 vs Wise $570** → 무역 거래는 Wise/WorldFirst
6. **Tarifflo HS분류 정확도 89%** > Zonos 44% → 외부 API 선택 시 참고
7. **EU CBAM 2026-01-01 발효** → 철강/알루미늄 수출 기업 즉시 영향
8. **인도네시아 할랄 2026-10-17 의무화** → 동남아 확장 시 필수
9. **eBL 글로벌 11% 채택, HMM 선도** → 한국 수출 플랫폼 차별화 기회
10. **한국 캐리어 글로벌 API 미지원** → AfterShip 또는 자체 크롤링 필요
