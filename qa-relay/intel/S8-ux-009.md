# S8-ux-009: 키보드 네비게이션 분석 — 한국어 랜딩

## 유형: 접근성 리포트
## 페이지: whistle-ai.com/ko

## Tab 순서 (첫 20개 요소)
1. 로고 버튼 (Whistle AI)
2. 서비스
3. 요금제
4. 바우처
5. 제휴문의
6. 로그인
7. 무료 AI 분석 시작하기 (CTA)
8. EN (언어 전환)
9. 무료 수출 진단 받기
10. 바우처로 무료 이용
11. URL 입력
12. 이미지 업로드
13. 파일 드래그 영역
14. 무료 AI 분석 시작
15-20. 딜 타임라인 단계들

## 평가
- ✅ Tab 순서 논리적 — 네비→CTA→콘텐츠 순서
- ✅ 모든 버튼이 focusable
- ⚠️ outline-style이 none — focus-visible이 키보드 사용 시에만 적용될 수 있음
- ⚠️ Skip Navigation 링크 없음 (S8-bug-002)
- ⚠️ 파일 드래그 영역(div)이 Tab 순서에 포함 — div인데 focusable

## 제안
- Skip Navigation 링크 추가 (Tab 1번)
- 파일 드래그 div에 role="button" 또는 tabindex 제거
- 포커스 링을 focus-visible이 아닌 focus에서도 키보드 사용자에게 표시되도록 확인 필요
