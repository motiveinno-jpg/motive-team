## 6. 무역 규제 변화 (2025-2026) — 최신

### US De Minimis 폐지 (확정)
| 항목 | 내용 |
|------|------|
| **현황** | 2025.08.29~ **전면 폐지** (모든 국가) |
| **2026.02.20** | 폐지 지속 확인 (행정명령) |
| **한국 영향** | 15% flat tariff (KORUS FTA 기반) |
| **10% Section 122** | 2026.02.24~ 모든 수입품 추가 10% |
| **기회** | 중국 직구(Temu/Shein) 가격경쟁력 약화 → 한국 제품 상대적 유리 |

### EU CBAM
- 2026.01~ 본격 부과 (철강, 알루미늄, 시멘트 등)
- 한국 영향: 연간 ~$5억 추가 비용 (철강 중심)

### FTA 현황
- 발효 22개 FTA / 59개국

- RCEP 활용률: 15-20% (아직 낮음)

## 7. 휘슬 AI 전략적 시사점

### TAM/SAM 업데이트
| 시장 | 규모 | 휘슬 연관 |
|------|------|-----------|
| K-뷰티 수출 | $114억 | 핵심 타겟 — 인디브랜드 D2C |
| K-푸드 수출 | $136억 | 핵심 타겟 — 할랄/비건 인증 |
| 중소기업 수출기업 | 9만개사 | SAM |
| AI 미활용 기업 | 88-92% | **즉시 침투 가능 시장** |
| 수출바우처 예산 | 6,867억원 | 수행기관 등록 → 매출 직결 |

### 핵심 기회 5가지
1. **AI 활용률 8-12%** → 저비용 AI 도구 수요 폭발 직전
2. **수출바우처 6,867억** → 수행기관 등록으로 정부 예산 직접 연결
3. **미국 de minimis 폐지** → 관세/통관 자동화 기능 차별화
4. **K-뷰티 미국 추월** → 미국 시장 특화 기능 (FDA 규정 등)
5. **K-푸드 중동 +22.6%** → 할랄 인증 가이드 기능 추가 가치

### 위험 요소
- 미국 15% flat tariff + Section 122 10% = 한국 수출 비용 증가
- 정부 "수출비서" AI — 공공 경쟁자 등장 가능성
- CBAM → 철강/화학 수출기업 부담 (K-뷰티/K-푸드는 영향 적음)
