# S8-ux-001: 320px 모바일에서 히어로 텍스트 줄바꿈 어색

## 유형: UX 개선 제안
## 페이지: whistle-ai.com/ko
## 뷰포트: 320px

## 설명
320px 소형 모바일에서 히어로 H1 "사진 한 장이면 수출이 시작됩니다"가
"사진 한 장이 / 면" 형태로 줄바꿈되어 읽기 어색함.
H1 폰트 크기(58px → 모바일 추정 ~38px)가 320px에서는 여전히 너무 큼.

## 개선 제안
- 320px에서 H1 폰트 크기를 32px 이하로 조정
- 또는 `word-break: keep-all`로 한글 단어 단위 줄바꿈 보장
```css
@media (max-width: 375px) {
  h1 { font-size: 32px; word-break: keep-all; }
}
```
