# S8-ux-002: 쿠키 배너가 하단 CTA 버튼 가림

## 유형: UX 개선 제안
## 페이지: whistle-ai.com/ko
## 뷰포트: 모바일 전체

## 설명
쿠키 동의 배너가 하단에 고정되어 있어 "무료 수출 진단 받기", "바우처로 무료 이용" CTA 버튼과 겹침.
특히 375px 이하 모바일에서 CTA 버튼이 쿠키 배너에 완전히 가려져 터치 불가.

## 개선 제안
1. 쿠키 배너를 상단으로 이동 (Apple/Stripe 패턴)
2. 또는 쿠키 배너에 max-height 제한 + 작은 폰트 사용
3. 또는 배너 표시 시 하단 콘텐츠에 padding-bottom 추가
