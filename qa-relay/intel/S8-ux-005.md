# S8-ux-005: 로그인 폼 — 비밀번호 표시/숨기기 토글 없음

## 유형: UX 개선 제안
## 페이지: whistle-ai.com/app (로그인)

## 설명
비밀번호 입력 필드에 눈 아이콘(show/hide) 토글이 없음.
모바일에서 특히 비밀번호 입력 확인이 어려움.

## 개선 제안
Password 입력 필드 우측에 아이콘 토글 추가:
- 기본: type="password" (숨김), 눈 아이콘
- 클릭 시: type="text" (표시), 눈에 줄 아이콘
- aria-label="비밀번호 표시" / "비밀번호 숨기기" 토글
