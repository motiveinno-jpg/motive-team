# S8-ux-007: 쿠키 배너 디자인 불일치 — 페이지별 다름

## 유형: UX 개선 제안
## 페이지: 전체

## 설명
쿠키 동의 배너가 페이지별로 디자인/문구/버튼 스타일이 다름:

| 페이지 | 문구 | 버튼 |
|--------|------|------|
| /ko (제조사 랜딩) | "이 웹사이트는 서비스 개선 및 분석을 위해..." | 거부 / 동의 (파란) |
| /en (영어 랜딩) | "We use cookies to improve..." | Decline / Accept (파란) |
| /buyer (바이어 랜딩) | "더 나은 경험을 위해 쿠키를 사용합니다..." | 수락 (파란) / 거부 (테두리) |
| /app (로그인) | "This website uses cookies..." | Decline / Accept |

## 문제
1. 동일 한국어 사이트인데 쿠키 배너 문구가 다름 (/ko vs /buyer)
2. 버튼 순서가 다름 (거부/동의 vs 수락/거부)
3. 브랜드 일관성 저해

## 개선 제안
- 공통 쿠키 배너 컴포넌트를 만들어 전 페이지에서 동일하게 사용
- 언어에 따라 한/영만 전환
