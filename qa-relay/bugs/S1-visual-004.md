# [S1-한국제조사] 시각적 UX: 서류 카드 제목이 영어로만 표시
- 심각도: P2(불편)
- 뷰포트: 데스크톱 + 모바일 모두
- 페이지: /app → 서류 생성
- 문제: 서류 카드 제목이 "Proforma Invoice", "Commercial Invoice", "Packing List", "Certificate of Origin", "Sales Contract" 등 전부 영어. 한국어 부제("견적 송장", "통관 송장", "포장 명세" 등)가 작게 표시되지만, 김미라 대표(영어 전혀 안됨)가 보면 "이게 뭔 말이지?" 하고 당황할 것. 한국어가 먼저 보여야 함.
- 스크린샷: S1-vis-07-docs-mobile.png, S1-P3-docs.png
- 제안: 한국어 이름을 주 제목으로, 영어를 부제로 전환. 예: "견적 송장 (Proforma Invoice)"
