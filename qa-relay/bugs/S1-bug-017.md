# [S1-한국제조사] 제품 추가 모달 — 사이드바 클릭 차단
- 심각도: P2(불편)
- 발견 시각: 2026-03-29 R4
- 페이지: /app → 제품 관리 → 제품 추가 모달
- 재현 경로: 제품 관리 → "+ 추가" 클릭 → 모달 열림 → 사이드바 다른 메뉴 클릭 시도
- 기대 동작: 사이드바 메뉴 클릭으로 모달 닫히고 해당 페이지로 이동
- 실제 동작: modal-overlay가 전체 화면을 덮어 사이드바 클릭 불가. ESC나 오버레이 클릭으로 닫아야만 이동 가능.
- 상세: `<div id="pm" class="modal-overlay show">` 가 z-index로 사이드바를 가림
- 영향: 사용자가 모달에 갇힌 느낌. "취소"나 "닫기" 버튼을 찾아야 함.
- 제안: 모달 오버레이가 사이드바를 가리지 않도록 하거나, 사이드바 클릭 시 자동으로 모달 닫기
- 브라우저: Chrome (Playwright headless)
- 상태: OPEN
