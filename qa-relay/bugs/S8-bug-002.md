# S8-bug-002: Skip Navigation 링크 없음 — 전체 사이트

## 심각도: P2
## 페이지: 전체
## 유형: 접근성 (WCAG 2.4.1)

## 설명
키보드 사용자가 페이지 상단의 반복 네비게이션을 건너뛸 수 있는 "Skip to main content" 링크가 없음.
WCAG 2.4.1 (Bypass Blocks) Level A 요구사항.

## 수정 제안
```html
<body>
  <a href="#main" class="skip-link"
     style="position:absolute;top:-40px;left:0;background:#00D4FF;color:#000;padding:8px 16px;z-index:10000;transition:top 0.3s;">
    메인 콘텐츠로 건너뛰기
  </a>
  ...
  <main id="main">
```
CSS:
```css
.skip-link:focus { top: 0; }
```

## ✅ FIXED (2026-03-30)
- Skip to content 링크 추가 (whistle-app.html + buyer-app.html)
- 탭 포커스 시 화면 상단에 나타남, 클릭하면 main content로 이동
