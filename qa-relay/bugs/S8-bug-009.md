# S8-bug-009: 로그인 폼 — 에러 메시지 컨테이너 없음

## 심각도: P2
## 페이지: whistle-ai.com/app (로그인)
## 유형: 접근성 + UX

## 설명
로그인 실패 시 에러 메시지를 표시할 `role="alert"` 또는 `aria-live` 컨테이너가 DOM에 없음.
스크린리더 사용자가 로그인 실패 사실을 인지할 수 없음.

## 수정 제안
```html
<div role="alert" aria-live="polite" id="login-error" style="display:none; color: #ff5252;">
  <!-- 로그인 실패 시 메시지 표시 -->
</div>
```
JS에서 에러 발생 시:
```js
const errorEl = document.getElementById('login-error');
errorEl.textContent = 'Invalid email or password. Please try again.';
errorEl.style.display = 'block';
```
