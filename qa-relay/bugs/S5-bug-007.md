# [S5-동남아바이어] P2: 무역 용어 설명 툴팁 없음 — 초급자 장벽

- 심각도: P2 (영어 초급 바이어 UX 저하)
- 뷰포트: 모바일/데스크톱 공통
- 페이지: 전체 (특히 Product Search, Deal Room, Orders)

## 문제
FOB, CIF, MOQ, HS Code, DDP 등 무역 전문 용어에 대한 설명 툴팁이 없음.
번역은 12개 언어로 제공되지만, 용어의 **의미 설명**이 없음.

## Nguyen Thi Mai 관점
- "FOB $8.00"이 뭔지 모름 → 배송비 포함인지 아닌지?
- "HS B4201.31"이 뭔지 모름 → 관세 코드?
- "MOQ: 200" → Minimum Order Quantity인 건 알겠는데, 협상 가능한지?

## 영향을 받는 용어들
| 용어 | 출현 빈도 | 설명 필요도 |
|------|----------|-----------|
| FOB | 높음 | "출하항 인도 가격 — 배송비 별도" |
| MOQ | 높음 | "최소 주문 수량" |
| HS Code | 중간 | "국제 상품 분류 코드 — 관세 결정" |
| CIF | 중간 | "운임보험료 포함 가격" |
| DDP | 낮음 | "관세 포함 배송완료 가격" |
| RFQ | 높음 | "견적 요청" |

## 제안
1. 각 용어 옆에 ℹ️ 아이콘 추가 → 클릭/탭 시 한 줄 설명 툴팁
2. 설명도 다국어 적용
3. 예: `FOB $8.00 ℹ️` → 탭 시 "Factory price. Shipping cost not included."
