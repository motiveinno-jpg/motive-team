# [S1-한국제조사] 전자서명 — 서류 페이지에서 서명 관련 UI 없음
- 심각도: P2(불편)
- 발견 시각: 2026-03-29 R3
- 페이지: /app → 서류 생성
- 재현 경로: 서류 생성 페이지에서 "서명" 관련 버튼/링크 검색
- 기대 동작: 생성된 서류에 전자서명 버튼 또는 서명 요청 기능이 있어야 함
- 실제 동작: 서류 목록/생성 화면에서 "서명" 텍스트를 포함한 버튼/링크가 전혀 없음. /sign-document 페이지는 존재하지만 서류 플로우에서 연결되지 않음.
- 영향: 김미라 대표가 서류를 만들어도 서명을 할 수 없음. 수출 서류에 서명은 필수.
- 제안: 서류 생성 후 "서명하기" 버튼 추가. 서류 미리보기에서 바로 서명 가능하도록.
- 브라우저: Chrome (Playwright headless)
- 상태: OPEN
