# [S1-한국제조사] 분석 결과 빈 데이터 표현 6가지 혼재
- 심각도: P3(미미)
- 발견 시각: 2026-03-29 R8
- 페이지: /app → AI 수출 분석 → 분석 결과 상세
- 재현 경로: 기존 분석 결과(오설록 녹차 등) 상세 보기
- 기대 동작: 빈 데이터에 일관된 한국어 표현 사용 (예: "데이터 없음" 통일)
- 실제 동작: "없습니다", "No data", "데이터 없음", "정보 없음", "N/A", "준비 중" 6가지 패턴이 혼재. 김미라 대표 입장에서 "N/A"나 "No data"는 이해 불가.
- 제안: 한국어 "데이터 없음" 또는 "정보 준비 중"으로 통일. 영어 표현(N/A, No data) 제거.
- 브라우저: Chrome (Playwright headless)
- 상태: OPEN
