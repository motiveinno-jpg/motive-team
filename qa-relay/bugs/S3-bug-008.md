# [S3-독일제조사] 언어 자동 감지 — Accept-Language 헤더 미사용, navigator.language만 의존
- 심각도: P1
- 발견 시각: 2026-03-29 22:40
- 페이지: 전체 (/, /app, /app/buyer)
- 재현 경로: Accept-Language: de-DE 헤더로 접속 → 영어 표시
- 기대 동작: Accept-Language 헤더 또는 navigator.language 기반으로 독일어 자동 감지. 첫 방문 시 독일어 UI 표시.
- 실제 동작:
  1. 서버 사이드(Cloudflare) Accept-Language 감지 없음
  2. 클라이언트에서 navigator.language.split('-')[0]만 사용 (whistle-app.html:1350)
  3. "de"는 supported 목록에 있어 실제 독일 브라우저에서는 감지 가능할 수 있으나:
     a. whistle-main.html (메인 랜딩)은 감지 결과를 UI에 적용하지 않음 — 항상 영어
     b. whistle-landing.html은 ko vs en만 구분 — 독일어 감지 없음
  4. 감지 결과 localStorage 저장은 whistle-app.html에서만 발생
