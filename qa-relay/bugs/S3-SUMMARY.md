## 핵심 인사이트 (4차 기준)

1. **S6 수정으로 독일어 UX 대폭 개선** — 사이드바, 번역 키 노출 해결, 언어 선택 UI 추가
2. **번역 품질은 높지만 커버리지 부족** — 번역된 부분은 전문적("Ursprungszeugnis", "Begasungszertifikat"), 미번역 부분은 영어 또는 한국어 혼재
3. **서비스 기본 관점이 "한국 제조사 → 미국 바이어"로 하드코딩** — 비용 계산기, 인증, 예시 데이터 모두 한국 수출 중심. 독일→한국 시나리오 미지원
4. **EUR 통화 인프라는 코드에 있지만 UI 기본값이 KRW/USD** — 국가 설정에 따른 통화 자동 전환 필요
5. **EU 필수 요소(IBAN, VAT, EORI, ATLAS) 부재** — 독일 기업이 실제로 사용하기에는 EU 맞춤화 필요
6. **카테고리 확장 불일치** — 설정에는 Medical/Chemical 있으나 가입 폼에는 누락
7. **가장 큰 구조적 문제: 양방향 무역 미지원** — 현재 한국 수출 전용. 독일 수출 시나리오(독일→한국)를 위한 역방향 흐름 필요
