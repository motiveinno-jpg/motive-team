# [S4-미국바이어] 바이어 대시보드 리로드 시 제조사 앱으로 리다이렉트 | P1 | buyer-app.html

## 재현 경로
1. /app/buyer 에서 바이어 계정으로 로그인 성공
2. 바이어 대시보드 정상 표시됨
3. 페이지 리로드 (F5 / 브라우저 새로고침)
4. → /app?lang=ja (제조사 앱 로그인 페이지)로 리다이렉트

- 바이어 대시보드가 그대로 유지됨

## 실제
- 리로드 시 바이어 세션이 사라지고 제조사 앱 로그인으로 이동
- 바이어 유저인데 제조사 앱(/app)으로 튕김
- buyer-app.html 라인 2553-2554의 역할 체크가 관련될 수 있음
