# [S8-UX성능] 시각적 UX: 모바일 H1 "수출이 시작됩니다" 줄바꿈 어색
- 심각도: P2 (불편)
- 뷰포트: 모바일 375px
- 페이지: /ko 한국어 랜딩 히어로

## 문제
H1 "사진 한 장이면 수출이 시작됩니다"가 375px에서:
"사진 한 장이면 / 수출이 시작됩 / 니다" 형태로 3줄.
"됩니다"가 혼자 아래로 내려가서 읽기 어색.
폰트 크기가 모바일에서도 너무 큼 (추정 ~42px).

## 제안
- word-break: keep-all 적용 (한글 단어 단위 줄바꿈)
- 모바일 H1 폰트 크기를 34px 이하로 축소
- 이상적 줄바꿈: "사진 한 장이면 / 수출이 시작됩니다"
