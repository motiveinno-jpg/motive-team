# [S5-동남아바이어] P2: 터치 타겟 44px 미달 — 모바일 조작 불편

- 심각도: P2 (모바일 사용성 저하)
- 뷰포트: 모바일 375x812
- 페이지: 로그인/가입 + 대시보드 전체

## 문제
WCAG 2.2 최소 터치 타겟 44x44px 기준 미달 요소 다수.

## 특히 심각한 것
- **체크박스 13x13px** — 모바일에서 누르기 거의 불가능
- **Forgot password 15px** — 실수로 못 누름

## 제안
- 모든 버튼 `min-height: 44px` 적용
- 체크박스에 `padding` 영역 확대 또는 라벨 전체를 터치 영역으로
- 링크에 `padding: 12px` 추가
