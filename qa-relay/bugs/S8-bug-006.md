# S8-bug-006: 영어 랜딩 — 모든 HTML 랜드마크 부재 (심각)

## 심각도: P1
## 페이지: whistle-ai.com/en (global-landing.html)
## 유형: 접근성 (WCAG 1.3.1, 2.4.1)

## 설명
영어 랜딩 페이지에 nav, main, header, footer 시맨틱 랜드마크가 **하나도 없음**.
스크린리더 사용자가 페이지 구조를 전혀 파악할 수 없음.

## 현재 상태
- nav: 0 ❌
- main: 0 ❌
- header: 0 ❌
- footer: 0 ❌

## 한국어 랜딩과 비교
한국어 랜딩은 nav(1), main(1), footer(1)가 있음. 영어 랜딩만 누락.

## 수정 제안
global-landing.html에서:
1. 상단 네비게이션 → `<header>` + `<nav>` 태그로 감싸기
2. 본문 콘텐츠 → `<main>` 태그로 감싸기
3. 하단 푸터 → `<footer>` 태그로 감싸기

## ✅ FIXED (2026-03-30)
- global-landing.html: div → header[role=banner], 본문 → main[role=main], div → footer[role=contentinfo]
