# S8-bug-016: 일본어 가입폼 — "Contact Name" 라벨 번역 누락

## 심각도: P2
## 페이지: whistle-ai.com/app (일본어 회원가입)
## 유형: i18n

## 설명
일본어 가입폼에서 "Contact Name *" 라벨만 영어로 남아있음.
다른 필드는 모두 일본어로 정상 번역됨.

### 현재 상태 (일본어)
| 필드 | 라벨 | placeholder | 상태 |
|------|------|------------|------|
| 회사명 | 会社名 * | 株式会社モティブ | ✅ |
| 담당자명 | **Contact Name** * | 山田太郎 | ❌ 영어 |
| 이메일 | メールアドレス * | email@company.com | ✅ |
| 비밀번호 | パスワード * | 8文字以上（英数字+記号） | ✅ |
| 국가 | 国 * | 選択... | ✅ |
| 약관 | 利用規約 & プライバシーポリシー に同意します | — | ✅ |
| CTA | 無料で始める | — | ✅ |

## 원인 추정
`_T` 딕셔너리에 contact_name의 일본어 번역이 누락되었거나 키가 다름.

## 수정 제안
whistle-app.html의 i18n 딕셔너리에서 contact_name 라벨의 ja 번역 추가:
```js
contact_name: { ja: '担当者名', ... }
```
