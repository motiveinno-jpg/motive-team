# [S5-동남아바이어] 시각적 UX: 쿠키 배너 하단 네비 가림

- 심각도: P2 (체감 불편)
- 뷰포트: 모바일 375x812
- 페이지: 전체 (첫 접속 시)

## 문제 (사람의 눈 기준)
1. 쿠키 동의 배너가 화면 하단에 고정되어 **하단 네비게이션 바를 완전히 가림**
2. 모바일에서 하단 네비(Trang chủ, Tìm kiếm, Giao dịch, Trò chuyện, Hồ sơ)를 사용하려면 먼저 쿠키 배너를 닫아야 함
3. 쿠키 배너의 "Decline"/"Accept" 버튼이 33px로 터치 타겟 미달
4. 온보딩의 "Tiếp theo" 버튼도 가려짐 — Step 2에서 Next를 못 찾아서 스크롤해야 했음

## 스크린샷

## 제안
1. 쿠키 배너를 하단 네비 위에 배치 (z-index 조정)
2. 또는 쿠키 배너를 상단에 표시 (모바일 전용)
3. Decline/Accept 버튼 44px로 확대
