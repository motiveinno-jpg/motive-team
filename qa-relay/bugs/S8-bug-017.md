# S8-bug-017: 바이어앱 로그인 — 언어 선택기 없음 + 디자인 불일치

## 심각도: P2
## 페이지: whistle-ai.com/app/buyer (buyer-app.html)
## 유형: UX + i18n

## 설명
### 1. 언어 선택기 없음
제조사앱(/app)에는 13개 언어 선택 버튼이 있지만,
바이어앱(/app/buyer)에는 언어 선택기가 전혀 없음.
일본어 설정(localStorage)이 유지되고 있어 일본어로 표시되지만,
사용자가 직접 언어를 바꿀 방법이 없음.

### 2. 디자인 불일치
| 항목 | 제조사앱 | 바이어앱 |
|------|---------|---------|
| 카드 border-radius | 각진 사각형 | 둥근 사각형 (16px+) |
| 배경 | 다크 단색 | 글래스모피즘 |
| CTA 버튼 | 그라데이션 cyan | 그라데이션 cyan (유사) |
| 서브타이틀 | "수출 성공의 시작" | "BUYER PORTAL" (영어) |
| 언어 선택기 | ✅ 13개 | ❌ 없음 |

### 3. "BUYER PORTAL" 영어 표시
일본어 모드인데 "BUYER PORTAL"이 영어로 표시됨.
→ "バイヤーポータル" 또는 현지화 필요.

## 수정 제안
1. buyer-app.html에도 제조사앱과 동일한 언어 선택기 추가
2. "BUYER PORTAL" → _isKorean 분기 또는 T() 함수로 번역
3. 두 로그인 화면의 디자인 톤 통일 (같은 브랜드)
