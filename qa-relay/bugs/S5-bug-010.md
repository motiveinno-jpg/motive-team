# [S5-동남아바이어] P2: 샘플 주문 기능 불명확 — 소량 바이어 니즈 미충족

- 심각도: P2 (소량 바이어 핵심 니즈)
- 뷰포트: 모바일/데스크톱 공통
- 페이지: Product Search, Product Detail

## 문제
Nguyen은 5~10개 샘플을 먼저 주문하고 싶음.
제품 카드에서 "Inquire" 버튼은 있지만:
1. "Sample Request" 전용 버튼이 없음
2. MOQ 100 이하 제품 필터는 있지만 "샘플 가능" 필터는 없음
3. Samples 메뉴가 사이드바에 있지만, 제품 카드에서 직접 접근 불가

## Nguyen 관점
- "이 제품 5개만 사볼 수 있나?" → 어떻게 해야 하는지 모름
- Inquire 버튼을 눌러서 메시지로 요청? → 절차가 불명확
- 경쟁자(Alibaba)는 "Order Sample" 버튼이 있음

## 제안
1. 제품 카드에 "Request Sample" 버튼 추가 (Inquire 옆)
2. 제품 검색에 "Sample Available" 필터/배지 추가
3. Sample Request 시 수량 입력 필드 (1~10개)
