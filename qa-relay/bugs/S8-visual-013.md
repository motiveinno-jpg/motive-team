# [S8-UX성능] 시각적 UX: 가로 모드(landscape) 네비게이션 완전 붕괴
- 심각도: P2 (불편)
- 뷰포트: 812x375 (iPhone 가로)
- 페이지: /ko 한국어 랜딩

## 문제
iPhone 가로 모드(812x375)에서:
1. **네비게이션 메뉴** — 모든 텍스트가 2줄 줄바꿈:
   - "서비 / 스", "요금 / 제", "바우 / 처", "제휴문 / 의"
   - "로그 / 인" 버튼도 줄바꿈
   - "무료 AI 분석 시작하 / 기" CTA 버튼 2줄
2. **H1** — 화면 높이가 375px라 히어로 텍스트 대부분이 화면 밖
3. **쿠키 배너** — 화면 높이의 ~30% 차지, 메인 콘텐츠 거의 안 보임

## 원인
812px 폭에서 데스크톱 네비게이션이 표시되지만(768px 이상이므로),
375px 높이에서 각 메뉴 항목 영역이 좁아 텍스트 줄바꿈.

## 제안
- landscape에서도 모바일 네비(햄버거)를 유지: `@media (max-height: 500px) and (orientation: landscape)`
- 또는 네비 padding/font-size 축소
- 쿠키 배너를 가로모드에서 더 컴팩트하게
