# [S8-UX성능] 시각적 UX: 제조사앱 vs 바이어앱 로그인 화면 디자인 불일치
- 심각도: P2 (불편)
- 뷰포트: 데스크톱 1280px
- 페이지: /app vs /app/buyer

## 문제
같은 Whistle AI 브랜드인데 제조사/바이어 로그인 화면이 전혀 다른 디자인.
사용자가 "같은 서비스인가?" 혼란.
- 제조사: 직각 카드, 어두운 배경, 언어 선택기 3줄
- 바이어: 둥근 글래스 카드, 그라데이션 배경, 언어 선택기 없음

## 제안
- 두 로그인 화면을 하나의 디자인 시스템으로 통일
- 공통 auth 카드 컴포넌트 사용
- 차이점: 서브타이틀만 ("수출 플랫폼" vs "바이어 포털")
