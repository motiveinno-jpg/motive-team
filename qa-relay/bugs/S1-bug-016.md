# [S1-한국제조사] 이미지 업로드 분석 — 업로드 버튼/탭이 보이지 않음
- 심각도: P2(불편)
- 발견 시각: 2026-03-29 R3
- 페이지: /app → AI 수출 분석 → 새 분석
- 재현 경로: 수출 분석 → 새 분석 시작 → 이미지 업로드 탭 찾기
- 기대 동작: URL 입력 외에 "이미지 업로드" 탭/버튼으로 상세페이지 사진을 올려서 분석 가능
- 실제 동작: input[type="file"]은 HTML에 존재하지만 보이는 "이미지 업로드" 버튼/탭이 없음. 사용자가 이미지 업로드 방법을 찾을 수 없음.
- 영향: 랜딩 페이지에서는 "📷 이미지 업로드" 버튼이 있는데, 앱 내부 분석에서는 보이지 않음. 일관성 부족.
- 브라우저: Chrome (Playwright headless)
- 상태: OPEN
