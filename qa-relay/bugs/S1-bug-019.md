# [S1-한국제조사] 수출 비용 계산기 — 극단값 입력 시 NaN/Infinity 표시
- 심각도: P1(핵심기능장애)
- 발견 시각: 2026-03-29 R6
- 페이지: /app → 수출 비용 도구 → 원가산출
- 재현 경로: 수출 비용 도구 → 원가산출 탭 → 제조원가=0, 포장비=0, 마진율=100%, MOQ=999999 입력
- 기대 동작: 유효하지 않은 계산 결과에 대해 경고 메시지 표시 또는 "0원" 등 합리적인 값 표시
- 실제 동작: 화면에 NaN 또는 Infinity가 표시됨
- 영향: 김미라 대표가 실수로 0이나 큰 숫자를 입력하면 깨진 화면을 보게 됨. 서비스 신뢰도 하락.
- 제안: 입력값 범위 검증 (제조원가 > 0, 마진율 0~99% 등) + NaN/Infinity 대신 "유효하지 않은 값입니다" 메시지
