# S8-bug-013: 토스트 시스템 — aria-live 없음

## 심각도: P2
## 페이지: whistle-app.html (UI.toast)
## 유형: 접근성 (WCAG 4.1.3)

## 설명
토스트 알림(.toast-container)에 `aria-live`, `role="status"`, `role="alert"` 속성이 없음.
스크린리더가 동적으로 추가되는 토스트 메시지를 읽지 않음.

### 현재 CSS (line 156):
```css
.toast-container{position:fixed;top:16px;right:16px;z-index:2000;...}
```

## 수정 제안
toast-container에 aria-live 추가:
```html
<div class="toast-container" aria-live="polite" role="status"></div>
```

에러 토스트는 assertive로:
```js
// UI.toast 수정
if (type === 'err') {
  container.setAttribute('aria-live', 'assertive');
  container.setAttribute('role', 'alert');
} else {
  container.setAttribute('aria-live', 'polite');
  container.setAttribute('role', 'status');
}
```
