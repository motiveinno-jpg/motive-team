# [S4-미국바이어] 회원가입 후 제조사앱으로 리다이렉트 | P1 | buyer-app.html

## 재현 경로
1. https://whistle-ai.com/app/buyer 접속
2. Sign Up 탭 → 폼 작성 (Company, Name, Email, PW, Country, Interest)
3. Create Account 클릭
4. → `/whistle-app.htm` (제조사 앱 로그인 페이지)로 리다이렉트됨

## 기대
- 회원가입 성공 → 바이어 대시보드 표시
- 또는 이메일 인증 안내 페이지 표시

## 실제
- 회원가입 버튼 클릭 후 `/whistle-app.htm` (제조사 앱)으로 이동
- UI가 일본어(ログイン)로 표시됨
- DB에 유저가 생성되지 않음 (auth.users 조회 결과 없음)
- 바이어가 가입을 시도하면 제조사 앱으로 튕겨나감

## 근본 원인 추정
- 가입 요청이 서버에 전달되지 않았거나, 가입 후 리다이렉트 로직이 바이어/제조사를 구분하지 못함
- whistle-main.htm 라우터 프레임이 바이어 가입 후 경로를 잘못 처리
