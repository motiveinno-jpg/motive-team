# [S1-한국제조사] 쿠키 동의 배너가 로그아웃 버튼을 가림
- 심각도: P2(불편)
- 발견 시각: 2026-03-29 23:10
- 페이지: /app (whistle-app)
- 재현 경로: 로그인 → 쿠키 동의 배너 미클릭 → 사이드바 하단 "로그아웃" 클릭 시도
- 기대 동작: 로그아웃 버튼이 정상 클릭됨
- 실제 동작: cookie-consent-banner (z-index 높음)가 사이드바 하단 로그아웃 버튼을 가려서 클릭 불가
- 상세: Playwright 에러 `<p id="cookie-text"> from <div class="show" id="cookie-consent-banner"> subtree intercepts pointer events`
- 해결 제안: 쿠키 배너 z-index 조정하거나, 사이드바 위에 표시하지 않도록 처리
- 스크린샷: 텍스트 로그로 확인
- 브라우저: Chrome (Playwright headless)
- 상태: OPEN
