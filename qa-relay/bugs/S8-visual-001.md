# [S8-UX성능] 시각적 UX: 모바일 히어로 CTA 버튼 텍스트 줄바꿈으로 못 읽겠다
- 심각도: P2 (불편)
- 뷰포트: 모바일 375px
- 페이지: /ko 한국어 랜딩 히어로

## 문제
"무료 수출 진단 받기" 버튼이 375px에서 "무료 수출 진 / 단 받기"로 어색하게 줄바꿈.
"바우처로 무료 이용" 버튼도 좁아서 빡빡해 보임.
두 버튼이 나란히 있는데 높이가 다름 (왼쪽이 더 높음) → 정렬 안 맞음.

## 제안
- 모바일에서 버튼 2개를 세로로 쌓기 (flex-direction: column)
- 또는 버튼 텍스트를 줄여서 한 줄에 표시: "무료 분석" / "바우처"
