# [S8-UX성능] 시각적 UX: 모바일 "How It Works" 4단계 카드 텍스트 잘림
- 심각도: P3 (어색)
- 뷰포트: 모바일 375px
- 페이지: /ko 한국어 랜딩 (스크롤 ~5500px 부근)

## 문제
수출 프로세스 4단계 카드(AI 분석→알리바바 입점→바이어 매칭→견적·샘플·계약)가
모바일에서 2x2 그리드로 표시되는데, 각 카드의 설명 텍스트가
"제품 사진/URL 업 / 로드 / → 13개 섹 / 션 보고서"처럼 잘려서 읽기 힘듦.

## 제안
- 모바일에서 1열 세로 배치 (카드 당 충분한 폭)
- 또는 설명 텍스트를 2줄 이내로 축약
