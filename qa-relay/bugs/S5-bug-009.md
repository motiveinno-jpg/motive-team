# [S5-동남아바이어] P3: 검색 필터 — 모바일에서 8줄 차지, 제품 밀림

- 심각도: P3 (모바일 사용성)
- 뷰포트: 모바일 375x812
- 페이지: Product Search

## 문제
필터 영역이 8줄(All Categories, Min Price, Max Price, MOQ Min, MOQ Max, All Origins, Relevance, Currency) 차지.
→ 실제 제품 카드가 스크롤 없이 보이지 않음
→ 첫 화면에 제품이 0개 표시

## Nguyen 관점
- 검색 페이지 진입 → 필터만 보임 → "제품이 없나?" 혼란
- 아래로 스크롤해야 첫 번째 제품 카드 보임

## 제안
1. 필터를 "Filters ▼" 토글로 접고, 기본 접힌 상태
2. 또는 필터를 가로 스크롤 칩 형태로 변경
3. 이미 "Hide Filters" 기능 있지만 기본이 열림 상태 → 모바일에서는 기본 접힘으로
