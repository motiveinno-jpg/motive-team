# [S5-동남아바이어] P2: 거래(Deals) 탭명 + 버튼 텍스트 잘림

- 심각도: P2 (모바일 레이아웃 깨짐 + 번역 누락)
- 뷰포트: 모바일 375x812
- 페이지: /app/buyer Deals (Giao dịch)

## 문제 1: 탭명 영어
- "🚀 Active Deals" / "📋 Inquiries" → **영어 (번역 누락)**
- 이 탭은 번역 적용 안 됨

## 문제 2: 버튼 텍스트 잘림
- "🔍 Tìm sản phẩ..." → **텍스트 잘림** (375px 폭에서 버튼 2개가 한 줄에 들어가면서 잘림)
- "📋 Đăng yêu cầu sourcing" → **"sourcing" 영어 혼재**

## 제안
1. "Active Deals" → "Giao dịch đang hoạt động", "Inquiries" → "Yêu cầu"
2. 버튼을 세로 배치 또는 텍스트 축소
3. "sourcing" → "tìm nguồn cung" (원료 조달)
