# S8-bug-012: 모달 시스템 — role="dialog", aria-modal, 포커스 트랩 없음

## 심각도: P1
## 페이지: whistle-app.html (모든 모달)
## 유형: 접근성 (WCAG 2.4.3, 1.3.1)

## 설명
모달 시스템(UI.modal)이 시각적으로 잘 구현되어 있으나 접근성 속성이 전혀 없음.

### 현재 모달 HTML (line 914):
```html
<div class="modal-overlay show" id="..." onclick="if(event.target===this)App.closeModal(...)">
  <div class="modal">
    <div class="modal-h"><h3>Title</h3><button class="modal-close">✕</button></div>
    <div class="modal-b">Body</div>
  </div>
</div>
```

### 누락 항목
1. ❌ `role="dialog"` 없음
2. ❌ `aria-modal="true"` 없음
3. ❌ `aria-labelledby` 없음 (h3와 연결)
4. ❌ 포커스 트랩 없음 (Tab이 모달 밖으로 나감)
5. ❌ ESC 키 핸들러 확인 필요 (prompt 모달에만 Enter 처리 있음)
6. ❌ 모달 열릴 때 모달 내부로 포커스 이동 없음
7. ❌ 모달 닫힐 때 트리거 요소로 포커스 복귀 없음

### 긍정적
- ✅ 바깥 클릭으로 닫기 구현
- ✅ ✕ 닫기 버튼 있음
- ✅ 모바일에서 bottom sheet 스타일로 전환 (좋은 UX)

## 수정 제안
```js
// UI.modal 함수 수정
modal: function(o) {
  // 트리거 요소 저장
  const trigger = document.activeElement;

  return '<div class="modal-overlay show" id="'+o.id+'" ' +
    'role="dialog" aria-modal="true" aria-labelledby="'+o.id+'-title" ' +
    'onclick="if(event.target===this)App.closeModal(\''+o.id+'\')">' +
    '<div class="modal"><div class="modal-h">' +
    '<h3 id="'+o.id+'-title">'+U.esc(o.title)+'</h3>' +
    '<button class="modal-close" aria-label="닫기" onclick="App.closeModal(\''+o.id+'\')">&#10005;</button>' +
    '</div>...</div></div>';
}
```
