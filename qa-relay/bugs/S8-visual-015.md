# [S8-UX성능] 시각적 UX: 라이트 모드 — 쿠키 배너 다크 스타일 불일치
- 심각도: P3 (어색)
- 뷰포트: 데스크톱 1280px + 모바일 375px
- 페이지: /app 로그인 (라이트 모드)

## 문제
라이트 모드(data-theme="light")에서 페이지 전체는 흰색 배경인데,
하단 쿠키 배너만 여전히 다크 스타일(어두운 배경 + 흰 텍스트).
시각적 불일치가 두드러짐.

## 긍정적
- ✅ 로그인 카드, 입력 필드, 버튼 모두 라이트 모드 적용 잘됨
- ✅ 언어 선택기 버튼도 라이트 스타일 적용
- ✅ 모바일에서도 라이트 모드 정상 작동

## 제안
- 쿠키 배너에도 `html[data-theme="light"]` CSS 추가
- 또는 쿠키 배너를 테마와 무관한 중립 스타일로 변경
