# [S1-한국제조사] 시각적 UX: 로그인 화면에 12개 언어 버튼이 나열됨
- 심각도: P3(어색)
- 뷰포트: 모바일 375px
- 페이지: /app 로그인 화면
- 문제: 로그인 폼 아래에 한국어, English, 日本語, 中文, Tiếng Việt, ไทย, Deutsch, Français, Español, Português, Bahasa, Türkçe, العربية 총 13개 언어 버튼이 3줄에 걸쳐 나열됨. 로그인 화면의 핵심은 "이메일+비밀번호 입력"인데 아래 언어 버튼이 너무 많아서 산만함. 첫 방문 사용자가 "이건 뭘 선택하라는 거지?" 혼란.
- 스크린샷: S1-vis-04-login-mobile.png
- 제안: 자동 감지 언어를 기본값으로, 변경 필요 시 드롭다운이나 설정에서 처리. 로그인 화면에서는 현재 언어 + "Change Language" 링크 정도로 축소.
