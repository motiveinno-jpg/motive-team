# S8-bug-003: prefers-reduced-motion 미지원

## 심각도: P2
## 페이지: 전체
## 유형: 접근성 (WCAG 2.3.3)

## 설명
CSS에 `@media (prefers-reduced-motion: reduce)` 미디어 쿼리가 없음.
전정기관 장애가 있는 사용자에게 애니메이션이 어지러움을 유발할 수 있음.
랜딩 페이지에 다수의 scroll-based 애니메이션이 존재하므로 반드시 필요.

## 수정 제안
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
```
