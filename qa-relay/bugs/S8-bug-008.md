# S8-bug-008: 로그인 폼 — input에 label 미연결 (접근성)

## 심각도: P2
## 페이지: whistle-ai.com/app (whistle-app.html 로그인 화면)
## 유형: 접근성 (WCAG 1.3.1, 4.1.2)

## 설명
Email, Password 입력 필드에 시각적 라벨 텍스트("Email *", "Password *")는 표시되지만,
`<label for="...">` 연결이 없고, `aria-label`도 없음.
스크린리더가 "edit text, email@company.com" (placeholder만) 으로 읽음.

## 현재 상태
- Email input: id 없음, label 미연결, aria-label 없음
- Password input: id 없음, label 미연결, aria-label 없음
- autocomplete="email" ✅ (올바름)
- autocomplete="current-password" ✅ (올바름)

## 수정 제안
```html
<label for="login-email">Email <span class="required">*</span></label>
<input type="email" id="login-email" name="email" placeholder="email@company.com" required autocomplete="email">

<label for="login-password">Password <span class="required">*</span></label>
<input type="password" id="login-password" name="password" placeholder="Password" required autocomplete="current-password">
```
