# [S5-동남아바이어] P2: KR 사용자 언어 강제 오버라이드 — 언어 변경 불가

- 심각도: P2
- 뷰포트: 공통
- 페이지: /app/buyer (loadUser 후)
- 코드 위치: buyer-app.html:2570-2572

## 문제
```javascript
_isKorean = (_authCountry==='KR');
if(!_isKorean && _userLang==='ko') { _userLang='en'; }
else if(_isKorean && _userLang!=='ko') { _userLang='ko'; }
```

이 로직 때문에:
- **country=KR인 사용자는 한국어 이외 선택 불가** (영어 선택해도 ko로 강제)
- Profile에서 English 선택 → 로그인 시 ko로 되돌아감

## 영향
- 한국 거주 외국인 (재한 외국인) 바이어가 영어/일본어 등 사용 불가
- 언어 설정 자유도 제한

## 제안
이 강제 오버라이드 로직 제거. 사용자가 수동 선택한 언어를 존중.
