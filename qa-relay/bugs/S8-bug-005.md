# S8-bug-005: 파일 입력 요소 접근성 이름 없음

## 심각도: P2
## 페이지: whistle-ai.com/ko
## 유형: 접근성

## 설명
랜딩 페이지의 이미지 업로드용 `<input type="file">` 요소에 aria-label, label, title 속성이 모두 없음.
스크린리더가 이 요소의 목적을 전달할 수 없음.

## 현재 코드
```html
<input type="file" accept="image/*,.pdf,.csv,.txt,.xlsx,.xls" style="display: none;">
```

## 수정 제안
```html
<input type="file" accept="image/*,.pdf,.csv,.txt,.xlsx,.xls" style="display: none;" aria-label="제품 이미지 또는 파일 업로드">
```

## ✅ FIXED (2026-03-30)
- fileUp 입력에 aria-label 추가
