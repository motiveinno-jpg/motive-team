# [S5-동남아바이어] 시각적 UX: 검색 페이지 필터 모바일 빽빽함

- 심각도: P2 (체감 불편)
- 뷰포트: 모바일 375x812
- 페이지: Product Search (Tìm sản phẩm)

## 문제 (사람의 눈 기준)
1. **필터가 화면 절반 이상 차지** — 검색 진입 시 제품이 하나도 안 보임
   - 카테고리, 가격 Min/Max, MOQ Min/Max, 출처, 정렬, 통화 = 8개 드롭다운
   - 모바일 375px에서 모두 표시되면 스크롤 없이 제품 0개 보임
2. **"Ẩn bộ lọc" (필터 숨기기)** 토글이 있지만 기본 열림 상태
3. 데스크톱에서는 1줄 가로 배치로 깔끔하지만 모바일에서 세로 쌓임
