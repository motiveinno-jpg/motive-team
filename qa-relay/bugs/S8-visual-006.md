# [S8-UX성능] 시각적 UX: 쿠키 배너가 숫자 통계를 가림 (모바일)
- 심각도: P2 (불편)
- 뷰포트: 모바일 375px
- 페이지: /ko, /en 모두

## 문제
첫 방문 시 하단 쿠키 배너가 "13개 분석 섹션 / 2분 분석 완료 / 최대 85%" 등
핵심 통계 숫자를 완전히 가림. 사용자가 이 영역을 못 보고 지나칠 수 있음.
영어 랜딩에서도 "13 / 60s / 200+ / 14" 숫자가 가려짐.

## 제안
- 쿠키 배너를 상단으로 이동 (많은 SaaS가 채택)
- 또는 하단 배너 높이를 줄이고 한 줄 표시
- 또는 바닥에 padding-bottom 추가하여 콘텐츠가 위로 올라가게
