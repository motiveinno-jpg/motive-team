# [S8-UX성능] 시각적 UX: 서비스 프리뷰 슬라이드 "프로젝트 자동 생성" 우측 빈 공간
- 심각도: P3 (어색)
- 뷰포트: 데스크톱 1280px
- 페이지: /ko 한국어 랜딩 — 서비스 프리뷰 (스크롤 ~7500px)

## 문제
"프로젝트 자동 생성" 슬라이드에서 단계 목록(기획~정산)이 좌측에만 있고,
카드 우측 절반이 완전히 빈 공간. 다른 슬라이드는 콘텐츠가 꽉 차있어서 이 슬라이드만 허전해 보임.

## 제안
- 우측에 프로젝트 카드 프리뷰 이미지 추가
- 또는 단계 목록을 2열 그리드로 배치하여 공간 활용
