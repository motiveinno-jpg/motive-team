# [S5-동남아바이어] P2: 통화 VND 미표시 — USD 고정

- 심각도: P2 (베트남 바이어 혼란)
- 뷰포트: 모바일 375x812
- 페이지: /app/buyer 대시보드

## 문제
베트남 바이어(country=VN, preferred_currency=VND)로 접속해도
대시보드에 "$33,500.00" (USD)로 표시.

VND로 표시되면 "₫851,900,000" (약 8.5억 동) — 큰 숫자 처리 필요.

## 체크 필요
1. preferred_currency가 UI에 반영되는 로직이 있는지
2. VND 큰 숫자(1 USD ≈ 25,400 VND)가 카드/테이블 레이아웃을 깨는지
3. VND는 소수점 없음 — $99.99 같은 소수점 가격이 VND로 변환 시 정수 처리

## 제안
1. 사용자 preferred_currency 기반 자동 환율 변환
2. VND 표시 시 천 단위 구분자: 851.900.000₫ (마침표 구분, 베트남 관행)
3. 레이아웃: 큰 숫자 대비 font-size 축소 또는 약어 (851.9M₫)
