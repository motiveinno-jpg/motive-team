# [S1-한국제조사] 시각적 UX: 수출 비용 도구 — 입력 필드 라벨 없는 영역
- 심각도: P3(어색)
- 뷰포트: 데스크톱 1280px
- 페이지: /app → 수출 비용 도구
- 문제: 원가산출 탭에서 "제조원가(개당)", "포장비(개당)", "마진율", "MOQ(수량)", "바이어 판매가" 필드가 세로로 나열됨. 각 필드에 placeholder 숫자만 있고, 필드 간 관계(어떤 것을 입력해야 뭐가 계산되는지)가 불명확. 하단 HS코드 분류 결과 테이블은 정보 밀도 좋지만, 입력 영역과 결과 영역의 시각적 구분이 약함.
- 스크린샷: S1-vis-11-cost-desktop.png
- 제안: 입력 영역을 카드/박스로 묶고 "① 원가 입력 → ② 자동 계산 결과" 흐름을 시각적으로 표현
