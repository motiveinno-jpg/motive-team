# [S1-한국제조사] 명령 팔레트(Ctrl+K) 검색이 필터링 안됨
- 심각도: P3(미미)
- 발견 시각: 2026-03-29 R9
- 페이지: /app → Ctrl+K 명령 팔레트
- 재현 경로: Ctrl+K → "분석" 검색 / "서류" 검색 / "비용" 검색
- 기대 동작: 검색어에 맞는 결과만 필터링되어 표시
- 실제 동작: "분석", "서류", "설정", "비용", "바이어" 어떤 키워드를 입력해도 동일한 5개 메뉴(AI 수출상품분석, 프로젝트, 제품, 서류, 비용)가 표시. 검색 필터링이 작동하지 않음.
- 영향: 명령 팔레트의 빠른 탐색 목적 달성 불가. 그냥 메뉴 목록만 보여줌.
- 제안: 검색어로 메뉴 이름/설명을 필터링. "서류" 검색 시 "서류 생성"만 표시.
- 브라우저: Chrome (Playwright headless)
- 상태: OPEN
