# [S1-한국제조사] 영어 랜딩(/en)에서 한국어 복귀 버튼 없음
- 심각도: P2(불편)
- 발견 시각: 2026-03-29 R5
- 페이지: /en (영어 랜딩)
- 재현 경로: /ko 한국어 랜딩 → "EN" 버튼 클릭 → /en 영어 랜딩 → 한국어 복귀 시도
- 기대 동작: 영어 랜딩에도 "KO" 또는 "한국어" 버튼이 있어 한국어로 돌아갈 수 있음
- 실제 동작: 영어 랜딩 헤더에 언어 전환 버튼 없음. 한국어 사이트에는 "EN" 버튼 있지만, 영어 사이트에서 한국어로 돌아올 방법이 없음.
- 영향: 김미라 대표가 실수로 EN을 누르면 영어 사이트에 갇힘. 영어 전혀 안되므로 뒤로가기 외에 방법 없음.
- 제안: 영어 랜딩 헤더에 "KO 한국어" 버튼 추가
