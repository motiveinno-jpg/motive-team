# S8-bug-007: 영어 랜딩 — 햄버거 메뉴 버튼 aria-label 없음

## 심각도: P2
## 페이지: whistle-ai.com/en (global-landing.html)
## 유형: 접근성

## 설명
모바일 햄버거 메뉴 버튼에 텍스트도 aria-label도 없음.
스크린리더가 "button"이라고만 읽어 목적을 알 수 없음.

## 현재 코드
```html
<button style="width: 40px; height: 40px; border-radius: 10px; background: rgba(255, 255, 255, 0.06)...">
  <!-- SVG icon only -->
</button>
```

## 비교: 한국어 랜딩
한국어 랜딩의 같은 버튼에는 `aria-label="메뉴 열기"` + `aria-expanded="false"`가 있음.
영어 랜딩에서만 누락.

## 수정 제안
```html
<button aria-label="Open menu" aria-expanded="false" ...>
```
