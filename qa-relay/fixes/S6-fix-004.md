# S6-fix-004: S2-bug-003 수정 — 언어 선택기 추가

## 버그ID
- S2-bug-003: 로그인/회원가입에 일본어 등 다국어 선택 옵션 없음

## 수정파일
- whistle-app.html

## 수정내용
1. `switchAuthLang(lang)` 함수 추가 — 언어 전환 시:
   - `_userLang`, `_isKorean` 업데이트
   - localStorage 저장 (`_lang`, `whistle_lang`)
   - `document.documentElement.lang` 업데이트
   - `_T=null`로 딕셔너리 캐시 리셋 → `render()` 호출
2. `authLangSelector()` 함수 추가 — 13개 언어 버튼 UI 생성
   - ko, en, ja, zh, vi, th, de, fr, es, pt, id, tr, ar
   - 현재 선택 언어 하이라이트 (pri 색상)
3. `pgAuth()` 함수의 로그인/회원가입/비밀번호 초기화 화면 모두에 삽입
4. `App` 객체에 `switchAuthLang` 등록

## 영향범위
- /app 로그인/회원가입/비밀번호 초기화 화면
- 일본어/중국어/베트남어 등 비-ko/en 사용자가 첫 방문 시 언어 전환 가능
- 기존 기능에 영향 없음
