# S6-fix-001: S2-bug-001 + S2-bug-002 수정

## 버그ID
- S2-bug-001: Country 드롭다운에 "select_ellipsis" 코드 변수명 노출
- S2-bug-002: Country 라벨 소문자 "country"

## 원인
`_ML` 딕셔너리에 `select_ellipsis`와 `country` 키에 `en` 항목이 누락됨.
`T()` 함수가 `_ML[key][_userLang]||_ML[key]['en']||key`로 폴백할 때
`en` 키가 없어서 딕셔너리 키 이름 자체("select_ellipsis", "country")가 그대로 출력.

## 수정파일
- whistle-app.html (line ~3131, ~2239)

## 수정내용
- `select_ellipsis` 항목에 `en:'Select...'` 추가
- `country` 항목에 `en:'Country'` 추가

## 영향범위
- /app 회원가입 폼의 Country 드롭다운 placeholder 텍스트
- /app 회원가입 폼의 Country 라벨 텍스트
- 영어 사용자에게만 영향 (다른 언어는 이미 번역 있음)
