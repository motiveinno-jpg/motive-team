[7/19] P07 — Service Chain
-------------------------------------------
7단계 수출 자동화 파이프라인입니다.
AI 분석 → 인증 가이드 → 서비스 분기(알리바바닷컴 or 독립) → 바이어 매칭 → 14종 서류 → 원가계산 → 물류추적.
알리바바닷컴 API 없이도 독립 운영이 가능하고, 알리바바닷컴 고객은 모티브가 운영을 대행합니다.

-------------------------------------------
[8/19] P08 — 핵심 기능 6가지
-------------------------------------------
Whistle AI의 핵심 기능 6가지입니다.
1. AI 수출 분석 — URL 한 장으로 3분 완료, 13개 섹션 리포트
2. 14종 서류 자동생성 — PI/CI/PL 등 원클릭
3. 수출 비용 도구 5종 — HS코드, FTA, 원가, 물류비, 마진
4. AI 번역 채팅 — 바이어 응답 30분→3분
5. 7단계 물류 추적 — B/L 추적, 바이어 공유
6. 알리바바닷컴 총판 운영 — 고객은 Whistle만 확인하면 됩니다.
6가지 기능 모두 라이브 운영 중입니다.
